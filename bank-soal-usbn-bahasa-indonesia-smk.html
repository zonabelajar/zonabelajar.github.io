33. no. 14 tolong bantu aku bang/kak. Kisi kisi Usbn Smk. ​


~Jawab~

SPLDV

___________________

misal:

kaleng cat = x

kuas = y

•••••

• Eliminasi:

2x + 3y = 101.500 (×2)

x + 2y = 53.500 (×3)

_______________________

4x + 6y = 203.000

3x + 6y = 160.500

___________________(–)

x = 42.500

Diperoleh harga satu kaleng cat = Rp42.500

• Substisukan x:

x + 2y = 53.500

42.500 + 2y = 53.500

2y = 53.500 - 42.500

2y =11.000

y = 5.500

Diperoleh harga satu kuas = Rp5.500

•••••

Sehingga,

= 1 kaleng cat + 1 kuas

= 42.500 + 5.500

= Rp48.000,00

Jawaban: B.
