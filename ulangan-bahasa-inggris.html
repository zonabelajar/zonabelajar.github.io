28. Ulangan harian bahasa inggris​

6. in order to

9. in order to

Penjelasan:

Kalau ada subjek keterusannya pasti so that, kalau gak ada subjek (jadi cuma ada kata kerja, predikat, objek, keterangan) keterusannya in order to

Contoh so that

I need to work out so that I can lose some weight ("I" //aku// itu termasuk subjek dari kalimat tersebut)

Contoh in order to

Let's work out in order to stay healthy during this pandemic

median (nilai tengah) = ?
