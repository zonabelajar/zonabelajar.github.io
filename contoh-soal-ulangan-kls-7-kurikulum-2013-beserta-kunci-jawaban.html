Kabel Listrik dipasang agak kendur agar kabel tidak putus saat suhu turun dan terjadi penyusutan.Koefisien muai panjang besi sebesar 12 x 10⁻⁵/°C atau  1,2 x 10⁻⁶/°CPanjang Baja pada suhu 70°C adalah 40,04 cm

Pembahasan

Pemuaian

Saat benda dikenai kalor, maka akan terjadi pemuaian pada benda tersebut yang akan mengakibatkan pertambahan ukuran pada benda. dan sebaliknya saat kalor dilepaskan dari benda, akan terjadi penyusutan, besarnya pemuaian atau penyusutan tergantung pada koefisien muai benda dan perubahan suhu.

Pada Pemuaian Panjang berlaku persamaan :ΔL = Lo x α x Δt  Lt = Lo + ΔL  

atau  

Lt = Lo + (Lo x α x Δt)  

Keterangan :  

Lo = Panjang awal = m  

Lt = Panjang akhir = m  

ΔL = Pertambahan Panjang = m  

α = Koefisien muai panjang

Pada Pemuaian Luas berlaku persamaan :ΔA = Ao x β x Δt  

dan  

At = Ao + ΔA  

Atau  

At = Ao + (Ao x β x Δt)  

Keterangan :  

ΔA = Pertambahan Luas  

Ao = Luas awal  

At = Luas Akhir  

β = Koefisien Muai Luas  

β = 2 α  

Pada Pemuaian Volume berlaku persamaan :ΔV = Vo x γ x Δt  

dan untuk Volume akhir adalah :  

Vt = Vo + ΔV  

atau  

Vt = V0 + (V0 x γ x Δt)  

Keterangan  

ΔV = Pertambahan Volume  

γ = Koefisien Muai Ruang atau Volume  

besarnya γ = 3α  

Δt= Perubahan Suhu  

Vo= Volume awal  

Vt = Volume akhir  

Pengerjaan soal :

Soal Np 1 :

Pemasangan Kabel Listrik dibuat kendur agar saatterjadi penyusutan dimalam hari kabel tidak putus, seperti sudah dijelaskan diatas jika benda mendapatkan kalor akan mengalami pemuaian, dan jika benda melepaskan kalor (mengalami penurunan suhu) akan terjadi penyuutan. Perubahan panjang tersebut dirumuskan dalam persamaan :

ΔL = Lo x α x Δt

Keterangan :  

Lo = Panjang awal = m  

ΔL = Pertambahan Panjang = m  

α = Koefisien muai panjang =  

Soal No 2 :Diketahui :

t1 = Suhu Awal = 0°C

t2 = Suhu akhir = 100°C

Lo = Panjang awal besi = 200 cm

Lt = Panjang akhir besi = 200,24 cm

ΔL = Perubahan panjang besi = 200,24 cm - 200 cm = 0,24 cm

Ditanyakan :

α = Kofisien muai panjang besi = ...

Penyelesaian :

ΔL = Lo x α x Δt

0,24 = 200 x α x 100

0,24 = 2.000α

α = 0,24 : 2.000

α = (24 x 10⁻²) : (2 x 10³)

α = 12 x 10⁻⁵   = 1,2 x 10⁻⁶

Soal No 3 :Diketahui :

t1 = Suhu awal = 20°C

t2 = Suhu akhir = 70°C

Δt = Perubahan Suhu = 70 - 20 = 50°C

Lo = Panjang Baja mula - mula = 40 cm

α   = Koefisien muai baja = 0,00002/°C = 2 x 10⁻⁵/°C

Ditanyakan :

Lt = Panjang akhir Baja = ...

Penyelesaian :

Lt = Lo + (Lo x α x t)

Lt = 40 + (40 x 2 x 10⁻⁵ x 50)

Lt = 40 + (4000 x 10⁻⁵)

Lt = 40 x (4 x 10⁻²)

Lt = 40 + 0,04

Lt = 40,04 cm

Kesimpulan :

Kabel di pasang agak kendur agar tidak putus saat penyusutanKoefisien muai panjang besi adalah 12 x 10⁻⁵/°C atau 1,2 x 10⁻⁶/°CPanjang akhir baja adalah 40,02 cm

Pembahasan

Halo adik-adik! Balik lagi di Brainly!! Gimana, masih semangat belajar kah? Nah untuk pertanyaan di atas itu sedikit masuk ke materi tentang fungsi atau persamaan kuadrat. Yuukk langsung aja kakak jelasin materi kali ini, oke? Oke! Pertama-tama apa itu fungsi kuadrat? Kakak jawab yaaa! Suatu persamaan yang memiliki variabel dimana variabel tersebut mempunyai pangkat yang paling tinggi dua maka persamaan tersebut disebut sebagai fungsi kuadrat. Bentuk umum dari fungsi kuadrat adalah [tex]y = f(x) = ax^2 + bx + c[/tex], dengan keterangan x disebut sebagai variabel, kemudian a, b adalah koefisien dan c adalah suatu konstanta, serta nilai dari a tidak sama dengan 0. Oke langsung aja yukk disimak penjabaran jawaban soal kali ini.

Persamaan kuadrat : [tex]x^2 + 5x + 6 = 0[/tex]

Untuk p = 1, jika q = 6 maka pq = 6, diperoleh p + q = 1 + 6 = 7

Untuk p = 2, jika q = 3 maka pq = 6, diperoleh p + q = 2 + 3 = 5

Untuk p = 3, jika q = 2 maka pq = 6, diperoleh p + q = 3 + 2 = 5

Untuk p = 6, jika q = 1 maka pq = 6, diperoleh p + q = 6 + 1 = 7

Untuk p = -1, jika q = -6 maka pq = 6, diperoleh p + q = -1 + (-6) = -1 - 6 = -7

Untuk p = -2, jika q = -3 maka pq = 6, diperoleh p + q = -2 + (-3) = -2 - 3 = -5

Untuk p = -3, jika q = -2 maka pq = 6, diperoleh p + q = -3 + (-2) = -3 - 2 = -5

Untuk p = -6, jika q = -1 maka pq = 6, diperoleh p + q = -6 + (-1) = -6 - 1 = -7

Karena harus memenuhi pq = 6 kemudian karena juga harus memenuhi p + q = 5, maka berdasarkan tabel tersebut pada baris kedua didapat p = 2 dan q = 3 atau berdasarkan baris ketiga ditulis p = 3 dan q = 2 (dua hasil ini merupakan hasil yang sama). Sehingga didapat pemfaktorannya

[tex]x^2 + 5x + 6 = (x + 2)(x + 3)[/tex]

Dengan demikian akar-akarnya adalah x = -2 dan x = -3

Persamaan kuadrat : [tex]x^2 + x - 6 = 0[/tex]

Untuk p = 1, jika q = -6 maka pq = -6, diperoleh p + q = 1 + (-6) = 1 - 6 = -5

Untuk p = 2, jika q = -3 maka pq = -6, diperoleh p + q = 2 + (-3) = 2 - 3 = -1

Untuk p = 3, jika q = -2 maka pq = -6, diperoleh p + q = 3 + (-2) = 3 - 2 = 1

Untuk p = 6, jika q = -1 maka pq = -6, diperoleh p + q = 6 + (-1) = 6 - 1 = 5

Untuk p = -1, jika q = 6 maka pq = -6, diperoleh p + q = -1 + 6 = 5

Untuk p = -2, jika q = 3 maka pq = -6, diperoleh p + q = -2 + 3 = 1

Untuk p = -3, jika q = 2 maka pq = -6, diperoleh p + q = -3 + 2 = -1

Untuk p = -6, jika q = 1 maka pq = -6, diperoleh p + q = -6 + 1 = -5

Karena harus memenuhi pq = -6 kemudian karena juga harus memenuhi p + q = 1, maka berdasarkan tabel tersebut pada baris keenam ditulis p = -2 dan q = 3 (dua hasil ini merupakan hasil yang sama). Sehingga didapat pemfaktorannya

[tex]x^2 + x - 6 = (x + 3)(x - 2)[/tex]

Dengan demikian akar-akarnya adalah x = -3 dan x = 2

Persamaan kuadrat : [tex]x^2 - x - 6 = 0[/tex]

Untuk p = 1, jika q = -6 maka pq = -6, diperoleh p + q = 1 + (-6) = 1 - 6 = -5

Untuk p = 2, jika q = -3 maka pq = -6, diperoleh p + q = 2 + (-3) = 2 - 3 = -1

Untuk p = 3, jika q = -2 maka pq = -6, diperoleh p + q = 3 + (-2) = 3 - 2 = 1

Untuk p = 6, jika q = -1 maka pq = -6, diperoleh p + q = 6 + (-1) = 6 - 1 = 5

Untuk p = -1, jika q = 6 maka pq = -6, diperoleh p + q = -1 + 6 = 5

Untuk p = -2, jika q = 3 maka pq = -6, diperoleh p + q = -2 + 3 = 1

Untuk p = -3, jika q = 2 maka pq = -6, diperoleh p + q = -3 + 2 = -1

Untuk p = -6, jika q = 1 maka pq = -6, diperoleh p + q = -6 + 1 = -5

Karena harus memenuhi pq = -6 kemudian karena juga harus memenuhi p + q = -1, maka berdasarkan tabel tersebut pada baris kedua didapat p = 2 dan q = -3 atau berdasarkan baris ketujuh ditulis p = -3 dan q = 2 (dua hasil ini merupakan hasil yang sama). Sehingga didapat pemfaktorannya

[tex]x^2 - x - 6 = (x + 2)(x - 3)[/tex]

Dengan demikian akar-akarnya adalah x = -2 dan x = 3

Selаin zаt pewаrnа, pemаnis, dаn penyedаp khususnyа dаlаm mаkаnаn dаn minumаn, аdа pulа zаt-zаt dаlаm mаkаnаn dаn minumаn yаng hаrus dihindаri, misаlnyа аlkohol. Zаt-zаt yаng dаpаt menyebаbkаn orаng menjаdi ketаgihаn disebut zаt аdiktif.

a. Sifat Perkalian Bilangan Berpangkat
Jika m, n adalah bilangan bulat positif dan a bilangan bulat maka am x an = am + n.
Contoh Soal :

= 29

b. Sifat Pembagian Bilangan Berpangkat
Jika m, n bilangan bulat positif dan m > n dengan a bilangan bulat dan a bukan 0, maka am : an = am - n.

Jika a, m, dan n bilangan bulat maka (am)n = am x n.
Contoh Soal:

Simple Present Tense

Simple Present Tense adalah tenses yang digunakan untuk menyatakan kejadian yang berlangsung sehari-hari (Daily routines).  

Kata keterangan yang sering digunakan dalam simple present tense adalah:

• Every day

• Every week

• every month

• every year

• once a week

• twice a day

• three times a month

• three times a year

• always

• often

• usually

• seldom

• rarely

• dan lain lain

Pattern yang digunakan dalam simple present tense adalah:

(+) S + V1 atau S + V1s/es

(-) S + do/ does + V1

(?) Do/ Does + S + V1?

NB:

V1 dan Do untuk subject I, You, They, We

V1s/es dan Does untuk subject He, She, It  

Pembahasan

They play together.

They study together.

THINGS SHE DOES NOT LIKE

They write bad words on the desks.

During the class, they move around  and they make a lot of noise.
