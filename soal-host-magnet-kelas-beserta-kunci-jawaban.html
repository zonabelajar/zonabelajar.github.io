Fungsi (Invers)

Jika ada suatu fungsi berbentuk [tex]f(x) = \frac{ax + b}{cx + d}[/tex], maka rumus cepatnya:

[tex]\boxed{f^{-1}(x) = \frac{-dx + b}{cx - a} }[/tex]

=========================================

Fungsi f: R → R dinyatakan dengan [tex]f(x) = \frac{x - 3}{2x + 1}, x \neq - \frac{1}{2}[/tex]. Tentukan f⁻¹(4)!

Langkah Penyelesaian:

*Misalkan f(x) = y

[tex]y = \frac{x - 3}{2x + 1}\\\\ y(2x + 1) = x - 3\\\\2xy + y = x - 3\\\\2xy - x=-y-3\\\\x(2y - 1) = -y-3\\\\x = \frac{-y - 3}{2y-1} \\\\\boxed{x = f^{-1}(x) = \frac{- x - 3}{2x - 1} }[/tex]

*Substitusi nilai

[tex]f^{-1}(4) =  \frac{- x - 3}{2x - 1}\\\\f^{-1}(4) = \frac{- 4 - 3}{2(4) - 1}\\\\f^{-1}(4) = \frac{- 7}{7}\\\\f^{-1}(4) = - 1\\\\\therefore\boxed{ f^{-1}(4) = - 1}[/tex]

==================================

17. Tentukan kelas. subnet mask, jumlah network dan host jika dketahui IP Address network 172.168.10.0 /22 ! * Jawaban Anda​


Berdasarkan informasi IP Address network 172.168.10.0 /22, dapat dihitung informasi berikut ini:

- Kelas: Kelas B

- Subnet mask: 255.255.252.0

- Jumlah network: 4

- Jumlah host per network: 1022

Penjelasan:

- Kelas B memiliki 16 bit untuk network dan 16 bit untuk host

- Subnet mask 22 bit (diwakili oleh 255.255.252.0) menunjukkan bahwa 22 bit pertama digunakan untuk network, sehingga sisa 10 bit digunakan untuk host

- Dengan 22 bit untuk network, terdapat 2^2 = 4 network yang dapat dibentuk

- Dengan 10 bit untuk host, terdapat 2^10 - 2 = 1022 host yang dapat di-assign pada setiap network (2 host dikurangi untuk network address dan broadcast address)

23. Subnet 170.238.102.177/30jumlah Subnetjumlah host per Subnetblok Subnetip networkhost pertamahost terakhirip broadcastkelasip addresssubnetwork ke berapa?​


Jawaban:

Mari kita analisis subnet dengan alamat IP 170.238.102.177/30:

1. **Jumlah Subnet**: Subnet ini memiliki prefix /30, yang berarti hanya ada 2^2 = 4 subnet yang tersedia.

2. **Jumlah Host per Subnet**: Dalam subnet /30, dua alamat digunakan untuk jaringan dan broadcast, sementara dua alamat tersedia untuk host. Oleh karena itu, setiap subnet memiliki 2 host yang dapat digunakan.

3. **Blok Subnet**: Subnet ini menggunakan prefix /30, yang berarti blok subnetnya adalah 4 alamat berurutan. Dalam hal ini, blok subnetnya adalah 170.238.102.176/30 hingga 170.238.102.179/30.

4. **Alamat IP Jaringan**: Alamat IP jaringan untuk subnet ini adalah 170.238.102.176.

5. **Host Pertama**: Host pertama dalam subnet ini adalah 170.238.102.177.

6. **Host Terakhir**: Host terakhir dalam subnet ini adalah 170.238.102.178.

7. **IP Broadcast**: IP broadcast dalam subnet ini adalah 170.238.102.179.

8. **Kelas**: Alamat IP 170.238.102.177/30 tidak termasuk dalam kelas A, B, atau C, karena ia adalah bagian dari prefix /30 yang lebih spesifik. Kelas khusus ini digunakan untuk subnetting.

9. **IP Address**: Alamat IP adalah 170.238.102.177.

10. **Subnetwork Ke Berapa**: Dengan prefix /30, ini adalah subnet ke-1 dalam jaringan 170.238.102.176/30.

A. hari Rabu

B. hari Kamis

C. hari Sabtu

D. hari Minggu

Pembahasan:

Jumlah hari dari 2 Mei ke 17 Agustus:

2 Mei – 31 Mei = 29 hari

1 Juni – 30 Juni = 30 hari

1 Juli – 31 Juli = 31 hari

1 Agustus – 17 Agustus = 17 Agustus

Total hari= 29 + 30 + 31 + 17 = 107

Hari akan berulang setiap 7 hari, sehingga perlu dicari sisa dari pembagian 107 oleh 7, yaitu

\[ = \frac{107}{7} = 15 \; \textrm{sisa} \; 2 \]

Dua hari setelah hari Selasa adalah hari Kamis.

Jadi, apabila Hari Pendidikan Nasional pada tanggal 2 Mei adalah hari Selasa, HUT Kemerdekaan RI tanggal 17 Agustus pada tahun yang sama adalah hari Kamis.
