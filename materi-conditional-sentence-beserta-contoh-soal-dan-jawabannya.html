7. Jelaskan materi conditional sentence type 3!


digunakan ketika result/consequence (hasil) dari condition (syarat) tidak ada kemungkinan terwujudkarena condition-nya harus sudah dipenuhi di masa lalu. Bagian conditional clause (if + condition) menggunakan subordinate conjunction “if”dan past perfect tense, sedangkan bagianmain clause (result) menggunakan perfect modal (modal + have + past participle). Kalimat Syarat Hasilnya If + Past Perfect Tense Past Future Perfect Tense If + S + had + Verb 3 S + would + have + Verb 3 Atau bisa juga dibalik menjadi: Hasilnya Kalimat Syarat Past Future Perfect Tense If + Past Perfect Tense S + would + have + Verb 3 If + S + had + Verb 3 contoh kalimat: If I had known that you were in hospital, I would have visited you. (Jika saja aku tahu bahwa kamu ada di rumah sakit, aku akan mengunjungimu).Conditional sentence type 3 = past impossible/unreal
yaitu pengandaian yang tidak mungkin menjadi nyata karena berlawanan dengan kondisi di masa lampau
Aturan :
a. Main clause (induk kalimat) : Past future perfect tense
b. If clause (anak kalimat) : Past perfect tense
c. Really meaning : SImple past tense

for example
1. I would have called her if I had known her name
the fact : I didn't call her because I didn't know her name
2. If you had given her money, She would have gone to school
the fact : you didn't give her money so she didn't go to school

17. Jelaskan materi tentang conditional sentence type 2 dan berikan contohnya ! Terimakasih


Conditional sentence type 2 bisa disebut juga dengan present unreal/impossible yaitu pengandaian yang sulit menjadi kenyataan karena berlawanan dengan kondisi disaat sekarang.
Aturan
Main clause : Simple past future tense
If clause : Simple past tense
- Semua subject memakai were
Faktanya yaitu present continuous
Pola 1
1. If + past tense + past future
- If I knew it, I would tell you
the fact : I don't know it, so I don't tell you
Pola 2
2. Simple past future tense + If + Simple past tense
- She would tell you about it, If she knew it
the fact : She doesn't tell you about it, because she doesn't know it

Apabila If berada di depan maka faktanya yaitu menggunakan kata so. Dan apabila if berada di tengah maka faktanya menggunakan kata because.

Semoga membantu dan bermanfaatconditional sentence type 2 yg digunakan utk mengandaikan sesuatu yg tdk terjadi di waktu lampau ttp berefek pd saat ini. contoh: if i were a rich people, i would help you. (the fact: i am not a rich people, so i can't help you)

Conditional Type 1 = a
--------"-------- Type 2 = b
--------"-------- Type 3 = c

1a. If I win the lottery, I will buy a new house
b. If I won the lottery, I would buy a new house
c. If I had won the lottery, I would have bought a new house

2a. Yoga won't repeat the class if he pass the exam
b. Yoga wouldn't repeat the class if he passed the exam
c. Yoga wouldn't have repeated the class if he had passed the exam

3a. If my grandma know the secret, she will be shocked
b. If my grandma knew the secret, she would be shocked
c. If my grandma had known the secret, she would have been shocked

4a. If I follow my brother's advice, I won't fail
b. If I followed my brother's advice, I wouldn't fail
c. If I had followed my brother's advice, I wouldn't have failed

5a. He will become a success(ful) bussinessman if his bussiness is crowded
b. He would become a success(ful) bussinessman if his bussiness was crowded
c. He would have become a success(ful) bussinessman if his bussiness had been crowded

6a. If you are lucky, you will get a free ticket to Singapore
b. If you were lucky, you would get a free ticket to Singapore
c. If you had been lucky, you would have got/gotten a free ticket to Singapore

7a. I will scold my brother if he skips class
b. I would scold my brother if he skipped class
c. I would have scolded my brother if he had skipped class

8a. My mother will cook in the kitchen if there are much (yang betul sebenarnya pakai many) vegetable(s)
b. My mother would cook in the kitchen if there were much (many) vegetable(s)

b. I would wait for you if you came on time
c. I would have waited for you if you had come on time

b. If Jennie knew the reason, she would tell me first
c. If Jennie had known the reason, she would have told me first

22. Contoh soal kalimat conditional sentence type 2


If my father were rich, I would live in a big villa with swimming polo inside.

(Jika ayah saya kaya, saya akan tinggal di sebuah vila besar dengan kolam renang di dalamnya.)

Keterangan: Dalam kalimat pengandaian ini, if my father was rich merupakan dependent clause yang tidak akan memiliki makna jika berdiri sendiri, sedangkan i would live in a big villa with swimming polo inside merupakan independent clause yang tetap memiliki makna meskipun tidak diikuti dengan kalimat di depan.

34. 50 soal serta jawaban tentang conditional sentence ?


Conditional sentence is a sentence that expresses a condition. A condition is something that can only happen IF something else occurs. Conditional sentences are made up of a dependent clause and an independent clause joined to express said condition.

Pembahasan

Conditional Sentence Type 1: It is possible and also very likely that the condition will be fulfilled.

Conditional Sentence Type 2: It is possible but very unlikely, that the condition will be fulfilled.

Conditional Sentence Type 3: It is impossible that the condition will be fulfilled because it refers to the past.

Contoh soal tentang conditional sentences yaitu

Conditional sentence type 1:

1. If we ... (not/work) harder, we ... (not pass) the exam.

2. If she ... (not go) to the meeting, I ....(not go) either.

3. If the teacher ... (give) us lots of homework this weekend, I ... (not be) happy.

4. If the children ... (not eat) soon, they ... (be) grumpy.

5. If I ... (not go) to bed soon, I ... (be) tired in the morning.

6. If John .... (drink) too much coffee, he ... (get) ill.

7. If she ...(go) to the library, she ... (study) more.

8. If they ... (not/arrive) soon, we ... (be) late.

9. If you ... (buy) the present, I ... (wrap) it up.

10. If Lucy ... (not quit) her job soon, she ... (go) crazy.

Conditional sentence type 2:

1. If she ... (have) her laptop with her, she ... (email) me.

2. If Lucy ... (have) enough time, she ... (travel) more.

3. If I ... (want) a new car, I ... (buy) one.

4. If Jose ... (not speak) good French, he ...(not move) to Paris.

5. If the children ... (be) in bed, I ... (be able to) have a bath.

6. If you ... (not be) so stubborn, we ... (not have) so many arguments!

7. If you ... (arrive) early, it ... (be) less stressful.

8. If Julie ... (like) chocolate, I ... (give) her some.

9. If Luke ... (live) in the UK, I ... (see) him more often.

10. If we ... (not be) so tired, we ... (go) out.

Conditional sentence type 3:

1. If the students ... (not be) late for the exam, they ... (pass).

2. If the weather ... (not be) so cold, we ... (go) to the beach.

3. If the baby ... (sleep) better last night, I ...(not be) so tired.

4. If we ... (tidy) our flat, we .. (not lose) our keys.

5. If Luke ... (not send) flowers to his mother, she ... (not be) happy.

6. If Julie ... (not go) to Sweden, she ... (go) to Germany.

7. If we ... (not have) an argument, we ... (not be) late.

8. If I ... (not go) to the party, I ... (not meet) Amanda.

9. If the children ... (not eat) all that chocolate, they ... (feel) sick.

10. If she ... (study) Mandarin, she ... (go) to Beijing.
