18. jawaban bahasa inggris hal 133 kurikulum 2013


Narrative Text

Narrative text adalah teks yang bertujuan untuk menghibur pembacanya. Terdapat beberapa jenis narrative text, yaitu:

1. Fable

2. Legend

3. Fairy tale

4. Horror

5. Romance

6. Mystery

7. Myth

8. dan lain lain

Generic Structure dari Narrative Text adalah:

• Orientation : It is about the opening paragraph where the characters of the story are introduced.(berisi pengenalan tokoh, tempat dan waktu terjadinya cerita (siapa atau apa, kapan dan dimana)

• Complication : Where the problems in the story developed. (Permasalahan muncul / mulai terjadi dan berkembang)

• Resolution : Where the problems in the story is solved. Masalah selesai, secara baik “happy ending” ataupun buruk “bad ending”.

• Coda / reorientation (optional) – lesson from the story

Language Features dari Narrative Text adalah:

• Use Past tense (killed, drunk, etc)

• Use adverb of time (Once upun a time, one day, etc)

• Use time conjunction (when, then, suddenly, etc)

• Use specific character. The character of the story is specific, not general. (Cinderella, Snow White, Alibaba, etc)

• Use action verbs. A verb that shows an action. (killed, dug, walked, etc)

• Use direct speech. It is to make the story lively. (Snow White said,”My name is Snow White). The direct speech uses present tense.

Sifat-Sifat Bilangan Berpangkat
a. Sifat Perkalian Bilangan Berpangkat
Jika m, n adalah bilangan bulat positif dan a bilangan bulat maka am x an = am + n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 x 33 = 39!
Jawab:
36 x 33 = ( 3 x 3 x 3 x 3 x 3 x 3) x (3 x 3 x 3)
6 faktor 3 faktor
= (3 x 3 x 3 x 3 x 3 x 3 x 3 x 3 x 3)
(6 + 3) faktor
= 26 + 3
= 29

b. Sifat Pembagian Bilangan Berpangkat
Jika m, n bilangan bulat positif dan m > n dengan a bilangan bulat dan a bukan 0, maka am : an = am - n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 : 33 = 33!
Jawab :
36 : 33 = 36
33

= 36 - 3
= 33

c. Sifat Perpangkatan Bilangan Berpangkat
Jika a, m, dan n bilangan bulat maka (am)n = am x n.
Contoh Soal:
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa (43)2 = 46.
Jawab:
(43)2 = (43) x (43)
= (4 x 4 x 4) x (4 x 4 x 4)
3 faktor 3 faktor

= 43 x 2 = 46

d. Sifat Perpangkatan pada Perkalian
Jika a, b bilangan bulat dan m bilangan bulat positifmaka (a x b)m = am x bm.
Contoh Soal=

Jawab:
(32)3 = (3 x 2) x (3 x 2) x (3 x 2)

e. Sifat Perpangkatan pada Pembagian.
Jika a, b bilangan bulat (b bukan 0) dan m bilangan bulat positif, maka (a : b)m = am : bm.

31. bahasa inggris hal 91-92 kurikulum 2013


Buku Bahasa Inggris kelas 8 kurikulum 2013 halaman 91-92 berisi tabel yang harus diisi oleh siswa. Tabel yang harus diisi berkaitan dengan collecting information, di mana siswa harus mengumpulkan informasi di sekitarnya. Beri

Halaman 91 : write each classmates answer about what they are doing now!

Alifah : she is reading a book.Yulia : she is writing a short story.Bobi : he is sweeping the floor.Jacob : he is cleaning the whiteboard.Maria: sheis playing a piano.

Halaman 92 : describe your school and find out what they are doing!

Place : library
Person : Mr. Adrian
Activity : reading a book.Place : library
Person : Cindy
Activity : working on her homework.Place : canteen
Person : Hans
Activity : eating lunch.Place: field
Person : Jeremy
Activity : practicing basketball.Place : music room
Person : Vivian
Activity : practicing piano.

Pembahasan

Ketika menulis kalimat-kalimat pernyataan saat mengumpulkan informasi (collecting information), tense yang paling tepat untuk digunakan adalah present continuous tense.

Pattern atau pola yang digunakan untuk membuat kalimat present continuous adalah :

  (+) Subject + is/am/are + Verb-ing

  (-) Subject + is/am/are not + Verb-ing

  (?) is/am/are + Subject + Verb-ing ?

Present continuous tense adalah tense yang pada umumnya digunakan untuk menyatakan hal-hal berikut:

Keadaan di mana sebuah kejadian sedang berlangsung.
Contoh : Andy is walking his dog.
Andy sedang mengajak anjngnya jalan-jalan.
Keadaan di mana sebuah kejadian sudah direncanakan dengan pasti.
Contoh : The exam is going to start at 10 A. M. tomorrow.
Ujian akan dimulai pada pukul 10 pagi besok.
Keadaan di mana sebuah perubahan sedang berlangsung.
Contoh : We are getting older.
Kita (sedang) bertambah tua.

Berikut adalah beberapa contoh kalimat yang menggunakan present continuous tense dan artinya :

Mrs. William is teaching English.
Mrs. William sedang mengajar bahasa Inggris.Jonathan is doing the dishes.
Jonathan sedang mencuci piring.Hana and Tio are playing together.
Hana dan Tio sedang bermain bersama.
