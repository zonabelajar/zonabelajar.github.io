1. Tutorial membuat game tebak Kata dengan bahasa c++


Berikut programnya, untuk mengubah menjadi 1-100 kamu hanya perlu mengubah "rand()%100"
#include <iostream>
#include <cstdlib>
using namespace std;
int main()
{
    int arr[1];
    arr[0]=rand()%10;
    cout << "-----------Tebak Angka dari 1-10 (Cari angka X)--------\n";

    for (int i=2; i>=0; i--)
    {

        cout << "Kesempatan mu untuk menebak adalah sebanyak " << i+1 << "kali\n\nMasukkan Angka: ";
        int tebak;
        cin >> tebak;
        if (arr[i]==tebak)
        {
            cout << "Angka X adalah " << tebak << "\nTebakanmu benar!\n";
            return 0;
        }
        else if (i>0) cout << "\nAngka yang ditebak salah!\n" << endl;
        else cout << "\nKamu gagal menebak angka.\n";
    }
}

19. apa arti bahasa Indonesia SARANG-IL CHUNGHAHAE URI dan tebak bahasa apa ​


Mungkin maksud Anda "Uri-eui Saeng-il Chunghahae-yo!" (우리의 생일 축하해요!) => Selamat ulang tahunnya kita.

[tex]\:[/tex]

Alasan

Kalimat yang kakak tulis merupakan kalimat rumpang sehingga pengertiannya pun tidak bisa dimengerti seseorang.Seharusnya subjek di letakkan pertama dengan pola S-K-O-P, sedangkan subjek dikalimat yang kakak tulis berada di akhir => UriSarang-il adalah kalimat yang salah, karena partikel -il (일) untuk penanda topik misalnya ulang tahun, liburan, dll. Mungkin seharusnya saeng-il (ulang tahun)

[tex]\:[/tex]

Pembahasan

Kalimat "Uri-ui Saeng-il Chunghahae-yo!" secara bahasa memang berarti selamat ulang tahunnya kita, namun artinya tidak merujuk ke ulang tahun biasa, tapi lebih ke anniversary pernikahan atau anniversary pacaran yang diucapkan oleh subjek yang berulangtahun sendiri. Dalam kalimat tersebut ada beberapa partikel yang akan saya bahas penggunaanya, yaitu :

[tex]\:[/tex]

Partikel -eui (의)

Partikel ini secara umum digunakan untuk menunjukan kepunyaan subjek. Pada kalimat Saeng-il (생일) menunjukan kepunyaan subjek yaitu ulang tahunnya sehingga di kata uri (우리) ditambahkan partikel eui (의).

[tex]\:[/tex]

Partikel -il (일)

Partikel ini secara umum digunakan untuk menekankan topik yang ada. Seperti halnya pada kalimat tersebut, topik saeng (ulang tahun) harus ditekankan dengan penggunaan partikel il (일)

[tex]\:[/tex]

Partikel -yo (요)

Partikel ini secara umum digunakan untuk menekankan tata kesopanan pada kata yang kita gunakan. Kalau saya cenderung selalu menggunakan partikel ini untuk mentraslatekan bahasa Indonesia ke Korea agar memiliki nilai kesopanan. Contohnya pada kata Chunghahae-yo (축하해요) yang secara mendalam diartikan sebagai pengungkapan selamat secara sopan.

[tex]\:[/tex]

Online artinya dalam jaringan, namun sering disingkat daring.

36. tuliskan algoritma tebak angka dalam bahasa Indonesia​


Jawaban:

1. Mulai

2. Inisialisasi variabel angka_acak dengan bilangan bulat acak antara batas awal dan batas akhir (misalnya antara 1 hingga 100).

3. Inisialisasi variabel tebakan dengan nilai awal 0.

4. Tampilkan pesan "Selamat datang di permainan Tebak Angka!"

5. Tampilkan pesan "Tebak angka antara 1 hingga 100:"

6. Selama tebakan tidak sama dengan angka_acak, lakukan langkah berikut:

7. Baca input dari pengguna dan simpan dalam variabel tebakan.

8. Jika tebakan kurang dari angka_acak, tampilkan pesan "Tebakan terlalu rendah!"

9. Jika tebakan lebih dari angka_acak, tampilkan pesan "Tebakan terlalu tinggi!"

10. Jika tebakan sama dengan angka_acak, lakukan langkah berikut:

11. Tampilkan pesan "Selamat! Anda berhasil menebak angka yang benar."

12. Tampilkan pesan "Angka yang benar adalah: " + angka_acak

13. Ulangi langkah 6.

14. Selesai

Algoritma di atas adalah contoh sederhana dari permainan tebak angka. Permainan ini akan terus berlanjut hingga pengguna menebak angka yang benar. Setiap tebakan akan memberikan petunjuk apakah angka yang ditebak terlalu rendah atau terlalu tinggi. Jika tebakan benar, permainan akan berakhir dan menampilkan angka yang benar.
