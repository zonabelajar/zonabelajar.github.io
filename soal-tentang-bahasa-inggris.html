Barat: West

Barat Laut : NorthWest

Utara : North

Timur Laut : NorthEast

Timur : East

Tenggara: SouthEast

Selatan : South

Barat Daya : SouthWest

Menuju Ke Timur : Eastern

Menuju Ke Barat : Western

Menuju Ke Utara : Northern

Menuju Ke Selatan : Southern

20. soal-soal bahasa Inggris​


Jawaban:

1. Apakah kamu pernah melihat gambar di atas tersebut ? ( Iya : Yes ) ( Tidak : no )

2. Biasanya simbol atau lambang tersebut kita dapat menemukan nya di ? ( area larangan rokok : No smoking area )

3. Apakah kamu sering melihat orang yang merokok di sekitar kamu ? ( iya : Yes ) ( tidak : no )

4. Apakah mereka merokok dengan seenaknya di tempat publik ? ( Yes / No )

5. Apakah kamu merasa merokok baik untuk mereka ? ( Yes / No )

6. Apakah mereka sendiri tau bahwa merokok tidak baik untuk mereka sendiri ? ( Yes / No )

7. Apakah mereka tau bahwa merokok dapat berdampak bagi orang di sekitar nya juga ? ( Yes / No )

8. Apakah bahkan mereka peduli ( tentang no.7 ) ? ( Yes / No )

9. Apakah kamu memiliki pengalaman buruk dengan perokok ? ( Yes / No )

10. Apa yang mereka lakukan ?

11. Apa yang harus kita lakukan ?

12. Apa yang harus pemerintah lakukan ?
