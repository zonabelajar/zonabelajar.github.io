MATERI: MICROSOFT OFFICE (WORD)

1. Apa fungsi CTRL + C?

Fungsi CTRL + C (copy) adalah untuk menyalin huruf, file, dsb.

2. Apa fungsi CTRL + V?

Fungsi CTRL + V (paste) adalah untuk menempelkan hal yang sudah dicopy sebelumnya.

3. Apa fungsi CTRL + B?

Fungsi CTRL + B (bold) adalah untuk menebalkan huruf / angka.Contoh: ABC

4. Apa fungsi CTRL + I?

Fungsi CTRL + I (italic) adalah untuk memiringkan huruf / angka.Contoh: ABC

5. Apa fungsi CTRL + U?

Fungsi CTRL + U (underline) adalah untuk memberikan garis bawah dibawah huruf / angka.Contoh: ABC

6. Apa fungsi CTRL + P?

Fungsi CTRL + P menampilkan form untuk print dokumen word. Di CTRL + P juga ada opsi untuk mengatur ukuran kertas yang akan digunakan.

7. Apa fungsi CTRL + N?

Fungsi CTRL + N adalah untuk membuat dokumen baru.

8. Apa fungsi subscript?

Subscript berguna untuk membuat huruf / angka menjadi kecil dan berletak di bawah.Contoh: [tex]X_{2}[/tex]  (angka "2" berada di bawah)

9. Apa fungsi superscript?

Superscript berguna untuk membuat huruf / angka menjadi kecil dan berletak di atas.Contoh: [tex]X^{2}[/tex]  (angka "2" berada di atas)

10. Sebutkan guna Microsoft Word!

Membuat, mengedit dokumen.
