16. jika suku pertama barisan aritmatika adalah -2 dengan 3,Sn adalah jumlah n suku pertama barisan aritmatika tersebut,dan Sn-2=68,maka nilai n adalah


Jawaban: nilai n dari barisan aritmatika tersebut adalah 70.

Penjelasan: Barisan aritmatika adalah barisan yang setiap sukunya memiliki selisih yang sama. Jika kita diberikan suku pertama barisan aritmatika dan selisih barisan tersebut, maka kita dapat mencari nilai dari semua suku barisan tersebut dengan menggunakan rumus barisan aritmatika sebagai berikut:

an = a1 + (n-1) * d

Dimana an adalah suku ke-n dari barisan tersebut, a1 adalah suku pertama dari barisan tersebut, dan d adalah selisih barisan tersebut.

Dari soal di atas, kita diberikan suku pertama barisan aritmatika adalah -2 dengan selisih 3. Jadi, kita dapat menuliskan rumus barisan aritmatika tersebut sebagai berikut:

an = -2 + (n-1) * 3

Selanjutnya, kita diberikan bahwa Sn-2=68. Sn adalah jumlah n suku pertama barisan aritmatika tersebut, sehingga kita dapat menuliskan rumus Sn sebagai berikut:

Sn = a1 + a2 + a3 + ... + an

= -2 + (-2 + 3) + (-2 + 3 + 3) + ... + (-2 + 3 + 3 + ... + 3)

= -2 + 1 + 4 + ... + (-2 + 3(n-1))

= n - 2

Kemudian, kita dapat mengganti rumus Sn di atas dengan nilai Sn-2 yang diberikan, sehingga didapatkan:

Sn-2 = 68

n - 2 = 68

n = 70

Jadi, nilai n dari barisan aritmatika tersebut adalah 70.

Beda barisan aritmatika tersebut adalah b = 2. Nilai tersebut diperoleh dari perhitungan jumlah n suku barisan aritmatika. Simak pembahasan berikut.

Pembahasan

Diketahui pada ulangan matematika terdapat soal mengenai jumlah barisan aritmatika.

Pada berkas soal yang diterima Adam, rumus tidak tercetak sempurna sehingga hanya terbaca "Sn = n² + ".

Adam masih bisa menjawab soal beda barisan tersebut.

Ditanya: Beda barisan aritmatika tersebut

Jawab:

Jumlah n suku pertama deret aritmetika dinotasikan dengn Sn, dan dirumuskan sebagai berikut:

Sn = [tex]\frac{n}{2}[/tex] (2a + (n - 1)b)

Sn = U₁ + U₂ + U₃ + .... + Un

dengan:

S₁ = U₁ = a

b = U₂ - U₁ = U₃ - U₂ = ...

Dari soal diketahui Sn = n² + ... Misal bagian rumpang pada rumus diketahui sebagai x, maka Sn diperoleh

Sn = n² + x, dengan x adalah sembarang bilangan

Maka diperoleh S₁ dan S₂ adalah:

S₁ = 1² + x

S₁ = 1 + x

dan

S₂ = 2² + x

S₂ = 4 + x

Karena S₁ = U₁ = a dan S₂ = U₁ + U₂ maka diperoleh persamaan

S₁ = U₁ = a = 1 + x

S₂ = U₁ + U₂

4 + x = 1 + x + U₂

U₂ = 4 + x - (1 + x)

U₂ = 4 + x - 1 - x

U₂ = 3

Maka beda barisan tersebut adalah

b = U₂ - U₁

b = 3 - (1 + x)

b = 3 - 1 - x

b = 2 - x

Untuk mendapatkan nilai x, maka dimisalkan untuk S₃ diperoleh sebagai berikut

Sn = [tex]\frac{n}{2}[/tex] (2a + (n - 1)b)

S₃ = [tex]\frac{3}{2}[/tex] (2a + (3 - 1)b)

S₃ = [tex]\frac{3}{2}[/tex] (2a + 2b)

S₃ = [tex]\frac{3}{2}[/tex] 2(a + b)

S₃ = 3(a + b)

karena a = x + 1, maka diperoleh

S₃ = 3(x + 1 + b)

S₃ = 3x + 3 + 3b...............1)

Ingat! Sn = n² + x

Sn = n² + x

S₃ = 3² + x

S₃ = 9 + x .........2)

Subtitusikan pers 2) ke per 1) maka diperoleh

9 + x = 3x + 3 + 3b

9 + x - 3x - 3 = 3b

6 - 2x = 3b

b = [tex]\frac{6 - 2x}{3}[/tex]

b = 2 - [tex]\frac{2x}{3}[/tex].........3)

Karena di awal diperoleh b = 2 - x, maka subtitusikan nilai b ke dalam pers 3)

2 - x = 2 - [tex]\frac{2x}{3}[/tex]

2 - 2 = - [tex]\frac{2x}{3}[/tex] + x

0 = - [tex]\frac{2x}{3}[/tex] + [tex]\frac{3x}{3}[/tex]

0 = [tex]\frac{x}{3}[/tex]

x = 0

Diperoleh nilai x adalah 0. Sehingga nilai b menjadi

b = 2 - x

b = 2 - 0

b = 2

∴ Jadi beda barisan yang dijawab Adam adalah b = 2.
