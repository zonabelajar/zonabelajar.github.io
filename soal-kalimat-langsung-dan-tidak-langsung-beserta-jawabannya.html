PREFACE Past Continous :

Digunakan untuk menyatakan hal yang telah sedang terjadi di masa lalu.

Formnya :

(+) S + be (was/were) + Present Participle (V ing) (-) S + be (was/were) + not + Present Participle (V ing) (?) Be (was/were) + S + Present Participle (V ing) + ?

Note :

was : digunakan untuk subject I, she, he, it, dan objek tunggal. were : digunakan untuk subject you, they, we, dan objek jamak.

Untuk "Question Word", formnya :

5W1H + Be (was/were) + S + Present Participle (V ing) + ?

5W1H : What, Who, Where, When, Why, How. (Apa, Siapa, Dimana, Kapan, Mengapa, Bagaimana)

➡ Contoh :

I was learning English yesterday. (+)

[Saya sedang belajar bahasa Inggris kemarin. ]

I was not learning English yesterday. (-)

[Saya tidak sedang belajar bahasa Inggris kemarin. ]

Was I learning English yesterday?

[Apakah saya sedang belajar bahasa Inggris kemarin? ]
