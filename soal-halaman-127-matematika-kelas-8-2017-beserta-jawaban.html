15) Jika f(2x + 1) = (x – 12)(x + 13) maka nilai f(31) adalah …

Jawab

f(2x + 1) = f(31)

2x + 1 = 31

2x = 31 – 1

2x = 30

x = 15

Jadi

f(2x + 1) = (x – 12)(x + 13)

f(2(15) + 1) = (15 – 12)(15 + 13)

f(30 + 1) = (3)(28)

f(31) = 84

16) Misalkan f(x) adalah fungsi yang memenuhi  

(a) untuk setiap bilangan real x dan y, maka f(x + y) = x + f(y)

(b) f(0) = 2

Nilai dari f(2016) adalah …

Jawab

Jika y = 0, maka:

f(x + y) = x + f(y)

f(x + 0) = x + f(0) ⇒ f(0) = 2

f(x) = x + 2

Jadi  

f(2016) = 2016 + 2 = 2018

17) Diketahui fungsi bilangan real f(x) = [tex]\frac{x}{1 - x}[/tex], untuk x ≠ 1. Nilai dari f(2016) + f(2015) + … + f(3) + f(2) + f([tex]\frac{1}{2}[/tex]) + f([tex]\frac{1}{3}[/tex]) + … + f([tex]\frac{1}{2015}[/tex]) + f([tex]\frac{1}{2016}[/tex]) adalah …

Jawab

Untuk menjawab soal tersebut, kita pasangkan fungsi-fungsi dengan angka yang sama yaitu

f(2) + [tex]f(\frac{1}{2})[/tex]

= [tex]\frac{2}{1 - 2} + \frac{\frac{1}{2}}{1 - \frac{1}{2}} [/tex]

= [tex]\frac{2}{-1} + \frac{\frac{1}{2}}{\frac{1}{2}} [/tex]

= –2 + 1

= –1  

f(3) + [tex]f(\frac{1}{3})[/tex]

= [tex]\frac{3}{1 - 3} + \frac{\frac{1}{3}}{1 - \frac{1}{3}} [/tex]

= [tex]\frac{3}{-2} + \frac{\frac{1}{3}}{\frac{2}{3}} [/tex]

= [tex]-\frac{3}{2} + \frac{1}{2} [/tex]

= [tex]-\frac{2}{2}[/tex]

= –1

:

f(2015) + [tex]f(\frac{1}{2015})[/tex]

= [tex]\frac{2015}{1 - 2015} + \frac{\frac{1}{2015}}{1 - \frac{1}{2015}} [/tex]

= [tex]\frac{2015}{-2014} + \frac{\frac{1}{2015}}{\frac{2014}{2015}} [/tex]

= [tex]-\frac{2015}{2014} + \frac{1}{2014} [/tex]

= [tex]-\frac{2014}{2014}[/tex]

= –1

f(2016) + [tex]f(\frac{1}{2016})[/tex]

= [tex]\frac{2016}{1 - 2016} + \frac{\frac{1}{2016}}{1 - \frac{1}{2016}} [/tex]

= [tex]\frac{2016}{-2015} + \frac{\frac{1}{2016}}{\frac{2015}{2016}} [/tex]

= [tex]-\frac{2016}{2015} + \frac{1}{2015} [/tex]

= [tex]-\frac{2015}{2015}[/tex]

= –1

Jadi penjumlahan yang menghasilkan –1 adalah 2016 – 1 = 2015 pasang

(dikurang satu karena pasangan f(1) dan f(1/1) tidak kita hitung)

Sehingga hasil dari penjumlahan soal diatas adalah

= (–1) + (–1) + (–1) + … + (–1) ⇒ sebanyak 2015

= (–1) . 2015

= –2015

18) Untuk setiap bilangan bulat x didefinisikan dengan fungsi f dengan f(x) adalah banyaknya angka (digit) dari x, contoh: f(206) = 3 dan f(2016) = 4. Nilai f(2²⁰¹⁶) + f(5²⁰¹⁶) adalah …

Jawab

Kita lihat pola f(2ⁿ) + f(5ⁿ)  

Jika n = 1

f(2¹) + f(5¹) = f(2) + f(5) = 1 + 1 = 2

Jika n = 2

f(2²) + f(5²) = f(4) + f(25) = 1 + 2 = 3

Jika n = 3

f(2³) + f(5³) = f(8) + f(125) = 1 + 3 = 4

Jika n = 4

f(2⁴) + f(5⁴) = f(16) + f(625) = 2 + 3 = 5

Jadi berdasarkan pola diatas f(2ⁿ) + f(5ⁿ) hasilnya adalah (n + 1)

berarti

f(2²⁰¹⁶) + f(5²⁰¹⁶) = 2016 + 1 = 2017

11. terjemahan soal bahasa inggris halaman 127 kelas 10


GRAMMAR REVIEW

Bacalah kalimat tidak lengkap di bawah ini. Lengkapilah dengan menggunakan to be "am/are/is" atau "was/were".

Pembahasan :

1). PRESENT TENSE (masa sekarang)

* To be (am) digunakan apabila subject nya "I".

* To be (are) digunakan apabila subject nya you/they/we/plural subject.

* To be (is) digunakan apabila subject nya she/he/it/kata ganti orang ketiga tunggal.

2). PAST TENSE (masa lampau)

* To be (were) digunakan apabila subject nya you/they/we/plural subject.

* To be (was) digunakan apabila subject nya I/she/he/it/kata ganti orang ketiga tunggal.

Jadi, jawabannya adalah :

1). In 1945, they (WERE) 17, so they (ARE) 85 now.

2). Today the weather (IS) Cold, but Last monday it (WAS) terribly hot.

3). I (AM) very thirsty. Can somebody give me a glass of water, please?

4). The defiant freedom frighters (WERE) very brave. They fought till death to defend the city.

5). I (AM) happy with Surabaya now. Years ago, It (WAS) very dirty and messy.

6). Don't buy those weapons. They (ARE) dangerous and illegal.

7). Hey, I like your new hat. It fits you well, and when you wear it, it reminds me of the 1945 freedom fighters. (IS) it expensive?

8). This time seventy years ago my grandfather (WAS) in Surabaya Joining the militia to fight against the British Army.

9). Where (ARE) the veterans? I don't know. They (WERE) in the lobby of the hotel five minutes ago.

10). The generals and the veterans (ARE) happy and optimistic about us now. They know we (ARE) always busy with good activities and never think of using drugs. When we met them last year, they (WERE) pessimistic.

1. Diketahui segitiga KLM dengan panjang sisi-sisinya k, l, dan m.

Pernyataan berikut yang benar dari segitiga KLM adalah ....

A. Jika m² = l² + k², besar ∠K = 90°

B. Jika m² = l² − k², besar ∠M = 90°

C. Jika m² = k² − l², besar ∠L = 90°

D. Jika k² = l² + m², besar ∠K = 90° (Benar)

Diketahui :

Segitiga KLM dengan panjang sisi k, l dan m

Ditanya :

Pernyataan yang benar ?

Dijawab :

Lihat gambar ilustrasi

A. Jika m² = l² + k², besar ∠K = 90°

Apabila ∠K = 90° maka sisi miring adalah sisi k

maka menurut Rumus Pythagoras :

k² = l² + m² (Pernyataan salah)

B. Jika m² = l² − k², besar ∠M = 90°

Apabila ∠M = 90° maka sisi miring adalah sisi m

maka menurut Rumus Pythagoras :

m² = k² + l² (Pernyataan salah)

C. Jika m² = k² − l², besar ∠L = 90°

Apabila ∠L = 90° maka sisi miring adalah sisi l

maka menurut Rumus Pythagoras :

l² = k² + m² (Pernyataan salah)

D. Jika k² = l² + m², besar ∠K = 90°

Apabila ∠K = 90° maka sisi miring adalah sisi k

maka menurut Rumus Pythagoras :

k² = l² + m² (Pernyataan benar)

kelompok tiga bilangan berikut.

Kelompok bilangan diatas yang merupakan Triple Pythagoras

(i) 3, 4, 5    

sisi miring = 5

25 = 9 + 16

196 ≠ 194 (Tidak terbukti)

625 = 625 (Terbukti)

841 = 400 + 441

841 = 841 (Terbukti)

c = sisi miring

36 > 9 + 25

1024 > 256 + 576

(iv).  20 cm , 30 cm, 34 cm

c = 34cm

1156 < 400 + 900

Segitiga lancip, karena c² < a² + b²

GRAMMAR REVIEW
Bacalah kalimat tidak lengkap di bawah ini. Lengkapilah dengan menggunakan to be "am/are/is" atau "was/were".

Pembahasan :
1). PRESENT TENSE (masa sekarang)
* To be (am) digunakan apabila subject nya "I".
* To be (are) digunakan apabila subject nya you/they/we/plural subject.
* To be (is) digunakan apabila subject nya she/he/it/kata ganti orang ketiga tunggal.

2). PAST TENSE (masa lampau)
* To be (were) digunakan apabila subject nya you/they/we/plural subject.
* To be (was) digunakan apabila subject nya I/she/he/it/kata ganti orang ketiga tunggal.


Jadi, jawabannya adalah :
1). In 1945, they (WERE) 17, so they (ARE) 85 now.

2). Today the weather (IS) Cold, but Last monday it (WAS) terribly hot.

3). I (AM) very thirsty. Can somebody give me a glass of water, please?

4). The defiant freedom frighters (WERE) very brave. They fought till death to defend the city.

5). I (AM) happy with Surabaya now. Years ago, It (WAS) very dirty and messy.

6). Don't buy those weapons. They (ARE) dangerous and illegal.

7). Hey, I like your new hat. It fits you well, and when you wear it, it reminds me of the 1945 freedom fighters. (IS) it expensive?

8). This time seventy years ago my grandfather (WAS) in Surabaya Joining the militia to fight against the British Army.

9). Where (ARE) the veterans? I don't know. They (WERE) in the lobby of the hotel five minutes ago.

10). The generals and the veterans (ARE) happy and optimistic about us now. They know we (ARE) always busy with good activities and never think of using drugs. When we met them last year, they (WERE) pessimistic.

GRAMMAR REVIEW  (Simple Present dan Simple Past)

were, are is,was am were am, was are is was are, were are, are, were

PEMBAHASAN:

Simple Present Tense  

Simple present tense digunakan untuk menyatakan fakta /kebenaran umum (general truth), kebiasaan (habitual action), atau kejadian yang terjadi pada saat ini (present). Simple present tense paling sering digunakan dalam bahasa Inggris karena digunakan untuk menyatakan kegiatan sehari-hari.  

Formula:  

Verbal sentences:  

(+) Subject + Verb1 s/es  

(-) Subject + do/does + not + Verb1  

(?) Do/does + Subject + Verb1?  

Nominal sentence:  

(+) Subject + to be (is/am/are) + Object/Complement  

(-) Subject + to be (is/am/are) + Object/Complement  

(?) To be (is/am/are) +Subject + Object/Complement?  

Contoh:  

1. (+) Susi brings her dictionary  

  (-) Susi does not bring her dictionary  

  (?) Does Susi bring her dictionary?

2. (+) My mom is very diligent

   (-) My mom is not very diligent

   (?) Is My mom very diligent?

3. (+) My father is in the office

   (-) My father is not in the office

   (?) Is My father in the office?

4. (+) Ammar and Amira are very smart

   (-) Ammar and Amira are not very smart

   (?) Are Ammar and Amira very smart?

5. (+) Egra and Yudi are always friendly

     (-) Egra and Yudi are not always friendly

     (?) Are Egra and Yudi always friendly?

6. (+) Susi is very diligent

    (-) Susi is not very diligent

    (?) Is Susi very diligent?

Cara menambahkan akhiran s atau es pada kata kerja:

Akhiran -es ditambahkan pada kata kerja (verb) yang berhuruf akhir -ch (catches), -s (kisses), -sh (brushes), -x (fixes), -z (whizzes) Akhiran -es juga ditambahkan pada kata kerja (veb) yang berhuruf akhir -y yang didahului oleh huruf mati dan akan berubah menjadi -i jika mendapat tambahan akhiran -es tersebut. contoh: study=studies, carry=carries, try=tries. Sedangkan bila -y didahului oleh huruf hidup, maka kerja itu hanya akan mendapat akhiran -s. contoh :pay-pays, say=says Akhiran -es juga ditambahkan khusus bagi kata kerja do dan go. contoh: do=does, go=goes

Keterangan Waktu Simple Pesent Tense:

always = selalu every = tiap often = sering usually = biasanya seldom = jarang rarely = jarang- jarang generally = pada umumnya frequently = seringkali sometimes = kadang-kadang ever = pernah never = tak pernah

Simple Past Tense

Simple past tense digunakan untuk menyatakan suatu kejadian di waktu tertentu di masa lampau.

Time expression: yesterday, last two days, last year, this morning, a week ago, ...etc.

Formula:

Verbal sentences:

(-)   Subject + to be (was/ were) + O/C  

(?)  To be (was/ were) +Subject + O/C?

    (?) Was Susi an artist?

3. (+) Susi brought her dictionary  

    (-) Susi did not bring her dictionary  

    (?) Did Susi bring her dictionary ?

4. (+) My father drank a cup of coffee this morning

AB : BC : AC = 1 : 1 : √2

Pembahasan : 

No. 1.

a.  hipotenusa = √32
     sudut = 45°

    cara perbandingan
    a : h = 1 : √2
    a : √32 = 1 : √2
    a / √32 = 1 / √2
             a = √32 / √2
             a = √16
             a = 4

   cara pythagoras
   a² + a² = (√32)²
        2a² = 32
          a² = 32 / 2
          a² = 16
          a = √16
          a = 4

b.  panjang sisi = 72
     sudut = 45°

    cara perbandingan
    s : a = 1 : √2
    72 : a = 1 : √2
    72 / a = 1 / √2
           a = 72 × √2
           a = 72√2

   cara pythagoras
   72² + 72² = a²
      2 × 72² = a²
               a = [tex] \sqrt{ 72^{2} \times 2} [/tex]
               a = 72√2

c.  hipotenusa = 16 cm
     sudut = 60°

     h : b = 2 : √3
    16 : b = 2 : √3
    16 / b = 2 / √3
    16 × √3 = b × 2
              b =  [tex] \frac{16 \sqrt{3} }{2} [/tex]
              b = 8√3 cm

d.  tinggi = 17√2
     sudut = 30°

    c : t = 1 : √3
    c : 17√2 = 1 : √3
    [tex] \frac{c}{17 \sqrt{2} } = \frac{1}{ \sqrt{3} } [/tex]
              c = [tex] \frac{17 \sqrt{2} }{ \sqrt{3} } [/tex]
              c = [tex] \frac{17 \sqrt{2} }{ \sqrt{3} } \times \frac{\sqrt{3} }{\sqrt{3} } [/tex]
              c = [tex] \frac{17 \sqrt{6} }{3} [/tex]

  d : t = 2 : √3
  d : 17√2 = 2 : √3
  [tex] \frac{d}{17 \sqrt{2} } = \frac{2}{ \sqrt{3} } [/tex]
  d × √3 = 17√2 × 2
     √3 d = 34√2
          d = [tex] \frac{34 \sqrt{2} }{ \sqrt{3} } [/tex]
          d = [tex] \frac{34 \sqrt{2} }{ \sqrt{3} } \times \frac{\sqrt{3} }{\sqrt{3} } [/tex]
          d = [tex] \frac{34 \sqrt{6} }{3} [/tex]

e.  alas = 5
     sudut = 60°

     alas : a = 1 : 2
      5 : a = 1 : 2
     5 / a = 1 / 2
          a = 5 × 2
          a = 10

    a : b = 2 : √3
    10 : b = 2 : √3
   10 / b = 2 / √3
   2 × b  = 10 × √3
        2b = 10√3
          b = 10√3 / 2
          b = 5√3

f.  hipotenusa = 20
    sudut = 60°

   d : h = 1 : 2
   d : 20 = 1 : 2
   d / 20 = 1 / 2
         d = 20 / 2
         d = 10

   e : h = √3 : 2
   e : 20 = √3 : 2
   e / 20 = √3 / 2
   e × 2 = 20 × √3
       2e = 20√3
         e = 20√3 / 2
         e = 10√3

AB = [tex] \frac{18 \sqrt{2} }{ \sqrt{2} } [/tex]
