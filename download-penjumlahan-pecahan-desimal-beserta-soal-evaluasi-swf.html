10. apa itu soal Evaluasi…………​


Evaluasi adalah salah satu cara/sarana dalam proses untuk menilai apakah kompetensi yang dipelajari telah dipahami dengan baik atau tidak.

Jawaban:

Pengertian Evaluasi

Apakah evaluasi itu? Pengertian evaluasi dapat dijelaskan secara bahasa maupun secara harfiah. Secara bahasa, evaluasi berasal dari kata bahasa inggris “ evaluation ” yang artinya penaksiran atau penilaian. Secara harfiah, evaluasi adalah proses untuk menentukan suatu hal atau objek berdasarkan tertentu untuk mencapai tujuan tertentu.

Evaluasi adalah suatu kegiatan mengumpulkan informasi mengenai kinerja sesuatu (metode, manusia, peralatan), dimana informasi tersebut akan digunakan untuk menentukan alternatif terbaik dalam membuat keputusan.

Evaluasi pengukuran dan perbaikan suatu kegiatan, seperti membandingkan hasil kegiatan dan menganalisisnya.

Evaluasi pengukuran atau perbaikan dalam suatu kegiatan yang dilaksanakan, seperti membandingkan hasil kegiatan yang telah direncanakan. Dari situlah tujuan evaluasi tersebut, agar rencana-rencana yang telah dibuat dapat mencapai yang telah ditetapkan dan dapat terselenggarakan. Dapat juga diartikan bahwa hasil evaluasi itu sendiri untuk perencanaan kembali bekerja sebagai administrasi dan fungsi manajemen yang terakhir mengkombinasikan dan mengumpulkan data dengan standar yang sesuai dengan tujuan yang ingin dicapai.

Dari pengertian di atas dapat memastikan bahwa evaluasi adalah proses menentukan nilai untuk suatu hal atau objek berdasarkan tujuan-tujuan tertentu untuk menentukan. Dalam perusahaan, evaluasi dapat diartikan sebagai proses pengukuran akan keefektifan strategi yang digunakan dalam upaya mencapai tujuan perusahaan.

30. Apakah perbedaan type video AVI, MPEG, 3GP, FLV, dan SWF​


Jawaban:

1. MPEG (Moving Picture Expert Group)

Ukuran file MPEG lebih kecil dibandingkan dengan file AVI, karena file MPEG termasuk kepada jenis file video yang termasuk format video terkompresi.

2. 3GP (3rd Generation Partnership Project)

Ukuran file 3GP merupakan ukuran file video terkecil di antara format video di atas. Karena format file 3GP biasanya digunakan pada handphone.

3. SWF (Shock Wave Flash)

File video SWF merupakan tipe video yang bisa dibuka oleh macromedia flash (Adobe), ukuran filenya juga lebih kecil jika dibangdingkan dengan MPEG dan FLV. Hasil convert AVI ke SWF menyebabkan gambar video menjadi pecah.

4. FLV (Flash Video)

Ukuran filenya sebanding dengan ukuran file MPEG & AVI.

Penggunaan masing-masing format :

1. AVI

Format ini termasuk pada format viodeo yang tidak dikompresi. Format AVI merupakan standar microsoft dan windows sebagai platformnya. Video yang menggunakan format ini akan menghasilkan ukuran file yang sangat besar karena resolusi yang dipakai sesuai dengan resolusi asli sumber video. Format ini juga dipakai sebagai kualitas terbaik yang digunakan untuk menentukan hasil akhir dari sebuah video. Namun format ini hanya dapat dimainkan di komputer saja.

2. MPEG

Termasuk dalam format video terkompresi dan dijadikan untuk video yang disimpan pada disk. Resolusi video yang berformat MPEG mendukung resolusi setengah layar dan satu layar, tergantung versi MPEGnya.

3. MOV

Termasuk dalam format video terkompresi. MOV dibuat oleh APPLE komputer dan dijalankan pada platform macintosh, tetapi sekarang sudah bisa digunakan pada windows dengan menginstall CODEC quick time. MOV termasuk video yang ditujukan untuk online video, website yang berbasis multimedia dan CD-ROM. Salah satu keunggulan format MOV adalah dapat mendukung video yang interaktive yaitu virtual reality.

4. 3GP

Format video ini biasanya di hasilkan oleh handphone. Sedangkan untuk kamera digital tidak mendukung format ini dalam vitur videonya.

5. SWF

Adalah tipe video yang bisa dibuka dengan oleh macromedia flash.

6. FLV

Ekstensi file FLV digunakan untuk menyimpan file Macromedia Flash Player. Setelah Macromedia dibeli oleh Adobe, telah diubah namanya menjadi Adobe Macromedia Player. Ini adalah yang paling populer format multimedia informasi di internet. Vector dapat mentransfer gambar, video streaming, audio dan teks. Satu keuntungan besar bahwa ia memiliki bahasa pemrograman yang terdapat didalamnya dan pengembang dapat software kode sangat maju internet menggunakan aplikasi ini.

32. jelaskan perbedaan .fla .swf dan .exe


jelaskan perbedaan .fla .swf dan .exePendahuluan

Macromedia Flash adalah sebuah perangkat lunak editor mutimedia dan animasi yang dibuat untuk para pecinta desain dan animasi supaya bisa berkreasi dengan mudah dalam membuat aplikasi-aplikasi sederhana, animasi-animasi yang interaktif untuk halaman web, film animasi kartun sederhana, dan presentasi. Selain itu, program ini tidak membatasi pengguna jika mampu menggunakan fitur software ini secara optimal dalam penggunaan fasilitas menggambar dan kemampuan menggunakan bahasa pemrograman pada software macromedia Flash (action script) ini kita bisa membuat mini game yang bagus.  

Kelebihan dari macromedia Flash dibandingkan program editor animasi yang lain adalah:

1. mudah dipelajari untuk sekalipun untuk seorang pemula yang masih awam dengan desain dan animasi.

2. dapat dengan mudah dan bebas dalam menuangkan kreatifitasnya berkreasi membuat animasi dengan gerakan yang dapat diatur sesuai dengan flow yang diinginkan.

3. output filenya berukurannya kecil. Karena Flash menggunakan animasi dengan gambar yang berbasis vektor

4. Software Macromedia Flash dapat menghasilkan file bertipe (ektensi). FLA yang bersifat fleksibel, karena dapat dikonversikan menjadi file lain bertipe .swf, .html, .gif, .png, .exe, .mov.

Pembahasan

.fla adalah ekstensi file project dari macromedia flash yang utama.

.swf adalah hasil kompilasi dari file project macromedia flash yang bisa langsung dijalankan animasinya jika sudah terdapat plugin flash player di komputer.

.exe adalah hasil kompilasi dari file project macromedia flash yang berupa executable file yang kompatibel dengan OS windows dan bisa langsung di jalankan tanpa plugin flash sekalipun.

Kesimpulan

perbedaan .fla .swf dan .exe adalah .fla adalah file project, perlu dibuka di program flash dulu untuk membukanya. .swf adalah file animasi yang siap dijalankan tapi membutuhkan plugin flash player, sedangkan. exe adalah file executable windows yang bisa langsung dibuka.
