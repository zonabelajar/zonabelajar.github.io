15. pakai caranya soal matematika kelas 4 SD semester 2 tentang Segitiga​


Rumus keliling segitiga

Keliling segitiga =  a + b + c

Jawaban atas pertanyaan dari segitiga A-E

Pertama-tama, anggap

Kolom ke-2 = a Kolom ke-3 = b dan Kolom ke-4 = c

A. Diketahui = a = 16 cm, b = 15 cm, c =17 cm.

Ditanya = keliling =___?

Keliling segitiga = a + b + c

Keliling segitiga = 16 cm + 15 cm +17 cm = 48 cm.

B. Diketahui = b = 42 dm, c = 46 dm dan keliling = 125 dm.

Ditanya = a =___?

Keliling segitiga = a + b + c

125 dm = a + 42 dm + 46 dm

125 dm - 42 dm - 46 dm = a

83 dm - 46 dm = a

a = 37 dm.

C. Diketahui = a = 20 m, c = 25 m, Keliling = 67 m

Ditanya = b =___?

Keliling segitiga = a + b + c

67 m = 20 m + b + 25 m

67 m - 20 m - 25 m = b

47 m - 25 m = b

b = 22 m.

D. Diketahui = a = 23 cm, b = 26 cm, Keliling = 63 cm

Ditanya = c =___?

Keliling segitiga = a + b + c

63 cm = 23 cm + 26 cm + c

63 cm - 23 cm - 26 cm = c

c = 40 cm - 26 cm = 14 cm.

E. Diketahui = a = 29 mm, b = 32 mm, Keliling = 100 mm

Ditanya = c =___?

Keliling segitiga = a + b + c

100 cm = 29 cm + 32 cm + c

c = 100 cm - 29 cm - 32 cm.

c = 71 cm - 32 cm = 39 cm.

MATERI: MICROSOFT OFFICE (WORD)

1. Apa fungsi CTRL + C?

Fungsi CTRL + C (copy) adalah untuk menyalin huruf, file, dsb.

2. Apa fungsi CTRL + V?

Fungsi CTRL + V (paste) adalah untuk menempelkan hal yang sudah dicopy sebelumnya.

3. Apa fungsi CTRL + B?

Fungsi CTRL + B (bold) adalah untuk menebalkan huruf / angka.Contoh: ABC

4. Apa fungsi CTRL + I?

Fungsi CTRL + I (italic) adalah untuk memiringkan huruf / angka.Contoh: ABC

5. Apa fungsi CTRL + U?

Fungsi CTRL + U (underline) adalah untuk memberikan garis bawah dibawah huruf / angka.Contoh: ABC

6. Apa fungsi CTRL + P?

Fungsi CTRL + P menampilkan form untuk print dokumen word. Di CTRL + P juga ada opsi untuk mengatur ukuran kertas yang akan digunakan.

7. Apa fungsi CTRL + N?

Fungsi CTRL + N adalah untuk membuat dokumen baru.

8. Apa fungsi subscript?

Subscript berguna untuk membuat huruf / angka menjadi kecil dan berletak di bawah.Contoh: [tex]X_{2}[/tex]  (angka "2" berada di bawah)

9. Apa fungsi superscript?

Superscript berguna untuk membuat huruf / angka menjadi kecil dan berletak di atas.Contoh: [tex]X^{2}[/tex]  (angka "2" berada di atas)
