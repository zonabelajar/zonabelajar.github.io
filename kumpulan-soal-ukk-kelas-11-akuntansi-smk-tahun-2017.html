Penjelasan dengan langkah-langkah:

Diketahui

[tex]f(x) = 2x + 1[/tex]

dan

[tex]g(x) = {x}^{2} - 1[/tex]

a. f + g

[tex]f(x) + g(x) = (2x + 1) + ( {x}^{2} - 1) \\ = 2x + 1 + {x}^{2} - 1 \\ = {x}^{2} + 2x[/tex]

b. f - g

[tex]f(x) - g(x) = 2x + 1 - ( {x}^{2} - 1) \\ = 2x + 1 - {x}^{2} + 1 \\ = - {x}^{2} + 2x + 2[/tex]

c. f x g

[tex]f(x) \times g(x) = (2x + 1) \times ( {x}^{2} - 1) \\ = 2 {x}^{3} - 2x + {x}^{2} - 1 \\ = 2 {x}^{3} + {x}^{2} - 2x - 1[/tex]

d. f : g

[tex]f(x) \div g(x) = (2x + 1) \div ( {x}^{2} - 1) \\ = \frac{2x + 1}{ {x}^{2} - 1 } [/tex]

25. soal matematika kelas 11 smk, dimohon bantuan nya


Penjelasan dengan langkah-langkah:

1.

h(x) = f(g(x))

h(x) = f(2x-4)

h(x) = 5 + 2(2x - 4)

h(x) = 5 + 4x - 8

h(x) = 4x - 3

2.

h^(-1)(4x - 3) = x

misalkan

(4x - 3) = a

4x = a + 3

x = (a + 3)/4

Jadi,

h^(-1)(a) = (a + 3)/4

Ubah a jadi x,

h^(-1)(x) = (x + 3)/4

3.

k(x) = g(f(x))

k(x) = g(2x + 5)

k(x) = 2(2x + 5) - 4

k(x) = 4x + 10 - 4

k(x) = 4x + 6

4.

k^(-1)(4x + 6) = x

misalkan.

(4x + 6) = a

4x = a - 6

x = (a - 6)/4

Jadi,

k^(-1)(a) = (a - 6)/4

Ubah a jadi x,

k^(-1)(x) = (x - 6)/4
