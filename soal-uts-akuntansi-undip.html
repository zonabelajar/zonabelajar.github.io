a = 1/2 + 2√3/4

P/x + Q/(x + 1) + R/(x - 1)
= [P(x + 1)(x - 1) + Qx(x - 1) + Rx(x + 1)] / x(x + 1)(x - 1)
= [P(x^2 - 1) + Q(x^2 - x) + R(x^2 + x)] / x(x^2 - 1)
= [Px^2 - P + Qx^2 - Qx + Rx^2 + Rx) / (x^3 - x)
= [(P + Q + R)x^2 + (R - Q)x - P] / (x^3 - x)
= (5x + 3) / (x^3 - x)
1) Konstanta
-P = 3 => P = -3

2) koefisien x
R - Q = 5
R = 5 + Q

3) koefisien x^2
P + Q + R = 0
-3 + Q + (5 + Q) = 0
2Q = -2
Q = -1

R = 5 + Q
R = 5 + (-1)
R = 4

(x^2-4)(x+3)

Nilai b yg memenuhi adalah 2 shg dpt difaktorkan mnjdi
(x^2-4)(x-2)
Perhatikan faktor pmbilang dan penyebut masing2 mengandung ekspresi (x^2-4) shg dpt dicoret, sisanya adalah
(x+3)/(x-2) yg merupakan pecahan linear.

x³ + ax² - 4x - 12 = 0

x1 . x2 . x3 = -(-12) = 12

faktornya yaitu 2,-2, dan -3
sehingga akar-akar dari persamaan tersebut yaitu 2,-2, dan -3
maka

-3 = -a

maka
