Diketahui :

Jarak titik A dari sumber bunyi (rA) = 4 meter

Jarak titik B dari sumber bunyi (rB) = 9 meter

Intensitas bunyi di titik A = IA

Intensitas bunyi di titik B = IB

Ditanya : IA : IB

Jawab :

Rumus hubungan antara intensitas dengan jarak :

Pembahasan soal intensitas bunyi 1

IA rA² = IB rB²

IA 4² = IB 9²

IA 16= IB 81

IA /IB = 81/ 16

2. Titik P dan Q berturut-turut berjarak 4 m dan 6 m dari sebuah sumber bunyi yang sedang bekerja. Jika bunyi yang dihasilkan sumber bunyi tersebut sampai di P dan Q dengan intensitas berturut-turut sebesar IP dan IQ, perbandingan IP dan IQ adalah ….

A. 2 : 1

B. 2 : 3

C. 3 : 2

D. 4 : 9

E. 9 : 4

JAWAB

Intensitas berbanding terbalik dengan kuadrat jarak.

\frac { { I }_{ P } }{ { I }_{ Q } } =\frac { { { r }_{ Q } }^{ 2 } }{ { { r }_{ P } }^{ 2 } }

\frac { { I }_{ P } }{ { I }_{ Q } } =\frac { { 6 }^{ 2 } }{ { 4 }^{ 2 } }

\frac { { I }_{ P } }{ { I }_{ Q } } =\frac { 36 }{ 16 }

\frac { { I }_{ P } }{ { I }_{ Q } } =\frac { 9 }{ 4 }

5. rasio intensitas bunyi 90db terhadap intensitas bunyi 60db adalah ​


Untuk menghitung rasio intensitas bunyi antara 90 dB dan 60 dB, kita perlu menggunakan rumus:

r = 10^((dB2 - dB1)/10)

di mana r adalah rasio intensitas bunyi, dB1 adalah intensitas bunyi awal, dan dB2 adalah intensitas bunyi akhir.

Dalam hal ini, dB1 = 60 dan dB2 = 90. Maka, substitusikan nilai ke dalam rumus:

r = 10^((90-60)/10) = 10^3 = 1000

Jadi, rasio intensitas bunyi antara 90 dB dan 60 dB adalah 1000. Ini berarti bahwa intensitas bunyi pada 90 dB adalah 1000 kali lebih besar dari intensitas bunyi pada 60 dB.

Bunyi adalah hasil getaran suatu benda.  

Kekuatan bunyi dipengaruhi oleh : amplitudo dan jarak sumber bunyi dengan pendengar.  

Tinggi rendahnya bunyi pengaruhi oleh : Frekuensi

b.Gelombang bunyi dapat mengalami interferensi  

c.Gelombang bunyi dapat mengalami pembiasan.  

d.Gelombang bunyi dapat mengalami pelenturan.  

e.Gelombang bunyi dapat mengalami pemantulan.  

f.Gelombang bunyi dapat diresonansikan

Interferensi gelombang bunyi  

Bunyi kuat ( interferensi konstruktif ) terjadi ketika perpaduan kedua gelombang bunyi pada suatu titik adalah sefase atau memiliki beda lintasan jarak tempuh yang nilainya sama dengan kelipatan bulat dari panjang gelombang bunyi.  

Bunyi kuat Δs = nλ; n = 0, 1, 2, 3, . . .                                          

Bunyi lemah ( Interferensi destruktif ) terjadi jika kedua gelombang yang bertemu pada suatu titik adalah berlawanan fase atau memiliki beda lintasan,  

Bunyi lemah Δs = (n+0,5) λ ; n = 0, 1, 2, 3, . . .                              

Sumber – sumber gelombang bunyi

17. soal fisika tentang taraf intensitas bunyi


◾ Materi : Bunyi
◾ Sub materi : Taraf Intensitas
◾ Mapel : Fisika
◾ Kelas : XII SMA (KTSP)
◾ Keyword : Taraf Intensitas

ㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡㅡ

Diketahui :
TI1 = 120 dB
n1 = 5 sumber bunyi
n2 = 5 + 45 = 50 sumber bunyi
R1 = 10 m

Ditanya :
a) TI2,2 = ? -> R2 = 10 m
b) TI2,2 = ? -> R2 = 100 m

Penyelesaian :
Taraf intensitas merupakan tingkat kebisingan dari suatu sumber bunyi.

Cari taraf intensitas ketika ditambah 45 sumber bunyi
n2 = n1 + 45
n2 = 5 + 45
n2 = 50 sumber bunyi

TI2 = TI1 + 10 log n
TI2 = 120 + 10 log 50
TI2 = 120 + 10 (1,69) ... log 50 = 1,69
TI2 = 120 + 16,9
TI2 = 136,9
TI2 = 137 dB

a) Pada jarak 10 m

TI2,2 = TI2,1 - 20 log R2/R1
TI2,2 = 137 - 20 log 10/10
TI2,2 = 137 - 20 log 1
TI2,2 = 137 - 0
TI2,2 = 137 dB

b) Pada jarak 100 m

TI2,2 = TI2,1 - 20 log R2/R1
TI2,2 = 137 - 20 log 100/10
TI2,2 = 137 - 20 log 10
TI2,2 = 137 - 20
TI2,2 = 117 dB

Keterangan :
TI2,2 = Taraf Intensitas 50 buah sumber bunyi pada jarak kedua (R2)
TI2,1 = Taraf intensitas 50 buah sumber bunyi pada jarak pertama (R1)

18. Soal mengenai intensitas bunyi


BUNYI
• intensitas

Pa = 1,2 W. rca = __?
Pb = 0,3 W. rcb = __?

intensitas bunyi
I = P / A
I = P / (4π r²)

Ica = Icb
Pa / (4π rca²) = Pb / (4π rcb²)
Pa / rca² = Pb / rcb²
rca² / rcb² = Pa / Pb
(rca / rcb)² = Pa / Pb
(rca / rcb)² = 1,2 / 0,3

27. intensitas bunyi dari petir lebih .... daripada intensitas bunyi dari jangkrik​


Intensitas bunyi petir lebih sedikit daripada intensitas bunyi jangkrik. Intensitas bunyi berarti jumlah bunyi yang dihasilkan oleh sumber bunyi dalam waktu tertentu.

Pembahasan

Bunyi adalah suara yang dihasilkan oleh sesuatu (sumber bunyi) yang bersifat alami dan bunyi yang disengaja (buatan). Bunyi alam contohnya petir, jangkrik, kambing, hujan dan lain-lain. Sedangkan bunyi buatan merupakan bunyi yang dihasilkan dari tindakan manusia seperti bunyi bedug, terompet, gendang, seruling, dan lain-lain.

Intensitas bunyi berarti jumlah bunyi yang dihasilkan oleh sumber bunyi dalam waktu tertentu. Intensitas bunyi ditunjukkan dengan banyak, sedang atau sedikit bunyi yang dihasilkan.

Dalam membandingkan intensitas bunyi petir dan intensitas bunyi jangkrik, intensitas bunyi petir lebih sedikit daripada intensitas bunyi jangkrik. Sebab dalam waktu tertentu, petir berbunyi hanya sekali dan maksimal 3 kali dengan suara yang menggelegar. Sedangkan bunyi jangkrik dalam satu waktu lebih banyak jumlahnya. Jangkrik biasa hidup di sawah, kebun, atau daerah yang banyak ditumbuhi tanaman atau pepohonan. Umumnya jangkrik berbunyi di malam hari dengan suara yang khas.

38. kebutuhan menurut intensitas beserta contohnya


Berdasarkan intensitasnya, kebutuhan dapat dibedakan menjadi kebutuhan primer, kebutuhan sekunder, dan kebutuhan tersier.

1 ) Kebutuhan primer

Kebutuhan primer adalah kebutuhan yang harus dipenuhi untuk kelangsungan hidup manusia. Kebutuhan primer disebut juga kebutuhan pokok/ kebutuhan dasar/ kebutuhan mutlak. Jika kebutuhan primer tidak dipenuhi, kelangsungan hidup manusia akan terganggu. Contoh kebutuhan primer, antara lain makan, minum, pakaian, dan tempat tinggal untuk berlindung dan istirahat sehari-hari.

2 ) Kebutuhan sekunder

Kebutuhan sekunder adalah kebutuhan yang dipenuhi setelah kebutuhan pokok terpenuhi. Kebutuhan sekunder merupakan pelengkap kebutuhan pokok. Contoh kebutuhan sekunder, antara lain radio, perabot rumah tangga, pendidikan, tas, sepeda motor, meja, kursi, alat tulis, dan alat olah raga. Kebutuhan sekunder setiap orang dapat berbeda-beda tergantung tingkat ekonomi dan status sosial. Contohnya alat olahraga bagi golongan berpenghasilan tinggi merupakan kebutuhan sekunder, sedangkan bagi mereka yang penghasilannya rendah merupakan kebutuhan tersier.

3 ) Kebutuhan tersier/ lux

Kebutuhan tersier adalah kebutuhan barang mewah yang dipenuhi setelah kebutuhan primer dan sekunder terpenuhi. Pada umumnya, pemenuhan kebutuhan tersier dilakukan oleh orang-orang yang berpenghasilan tinggi dan biasanya digunakan untuk menunjukkan status sosial agar lebih dihargai oleh orang lain dan lebih terpandang. Contoh kebutuhan tersier, antara lain kebutuhan rumah mewah, perhiasan, berlian, dan mobil mewah.
