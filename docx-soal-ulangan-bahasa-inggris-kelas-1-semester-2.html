4. Apa yang dimaksud dengan docx?


Jawaban:

Sebuah file dengan ekstensi DOCX merupakan file Microsoft Word Open XML Format Document. Ya, ini merupakan file yang tercipta dari pembuatan dokumen di Microsoft Word, aplikasi pembuatan teks bikinan Microsoft. File dengan ekstensi DOCX berbasiskan XML dan bisa saja mengandung teks, objek, style, formating, dan gambar.

Sebuah file dengan ekstensi DOCX merupakan file Microsoft Word Open XML Format Document. Ya, ini merupakan file yang tercipta dari pembuatan dokumen di Microsoft Word, aplikasi pembuatan teks bikinan Microsoft.

File dengan ekstensi DOCX berbasiskan XML dan bisa saja mengandung teks, objek, style, formating, dan gambar. Semua komponen itu sebenarnya tersimpan dalam file yang bebeda, tetapi kemudian dipadatkan dalam sebuah file DOCX yang sudah terkompresi.

File berbasiskan DOCX baru saja hadir di Microsoft Word, tepatnya di Word edisi 2007. Versi Word lebih awal menggunakan ekstensi file DOC, tanpa “X”.

14. Jelaskan apa perbedaan ekstensi. Doc dengan docx dan.xls dengan .xlsx



Secara singkat, perbedaan antara format .doc dan .docx itu dari tahun produksinya. Sejak awal produksi, Microsoft Word menggunakan format *.doc, tapi format *.doc terkadang bermasalah kalau dibuka dengan software selain Microsoft Word, sementara tujuan dari microsoft adalah membuat format *.doc menjadi format universal bagi seluruh Software sejenis dengan Microsoft Word. jadi.. sejak Update Microsoft Word 2007, format standar yang digunakan oleh Microsoft adalah *.docx, karena selain bisa dibuka dengan software lain, *.docx juga memiliki fitur-fitur tambahan dan pengelolaan data yang lebih baik. Kekurangannya? format *.docx tidak bisa dibuka dengan Microsoft Word 2006 kebawah.

Hal yang sama berlaku untuk *.xls dan *.xlsx, hanya saja berlaku untuk Microsoft Excel dan sejenisnya. *.xls adalah *.doc, sedangkan *.xlsx adalah *.docx

Maaf kalau agak ribet :Dekstensi Doc dan docx sama - sama merupakan ekstensi dari microsoft word, perbedaannya adalah doc merupakan ekstensi default versi lama (ms2003) dan docx merupakan ekstensi default ms office versi 2007 - present.

sedangkan xls dan xlsx merupakan ekstensi dari microsoft excel, perbedaannya adalah doc merupakan ekstensi default versi lama (ms2003) dan docx merupakan ekstensi default ms office versi 2007 - present.
