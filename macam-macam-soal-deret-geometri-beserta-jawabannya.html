Jawaban:

Berikut ini contoh soal cerita deret geometri beserta jawabannya:

Soal:

Umi merencanakan untuk menabung uang setiap bulan. Pada bulan pertama, Umi menabung Rp50.000. Pada bulan berikutnya, jumlah uang yag ditabung bertambah 50% dari bulan sebelumnya. Jika Umi menabung selama 6 bulan, berapakah total uang yang sudah ditabung Umi?

Langkah penyelesaian:

1. Bulan ke-1: Rp50.000

2. Bulan ke-2: Rp50.000 + 50% dari Rp50.000 = Rp50.000 + Rp25.000 = Rp75.000

3. Bulan ke-3: Rp75.000 + 50% dari Rp75.000 = Rp75.000 + Rp37.500 = Rp112.500

4. Bulan ke-4: Rp112.500 + 50% dari Rp112.500 = Rp112.500 + Rp56.250 = Rp168.750

5. Bulan ke-5: Rp168.750 + 50% dari Rp168.750 = Rp168.750 + Rp84.375 = Rp253.125

6. Bulan ke-6: Rp253.125 + 50% dari Rp253.125 = Rp253.125 + Rp126.562,5 = Rp379.687,5

Total uang yang ditabung = Rp50.000 + Rp75.000 + Rp112.500 + Rp168.750 + Rp253.125 + Rp379.687,5

= Rp1.039.062,5

Jadi, total uang yang sudah ditabung Umi selama 6 bulan adalah Rp1.039.062,5

Bagaimana? Saya berikan contoh soal cerita deret geometri beserta langkah penyelesaiannya. Mohon dinilai apakah sudah sesuai dengan yang Anda minta.

soal 1: 4+ 8 + 12+ ....

rumus deret aritmatika:

[tex]S_n = \frac{n}{2}(2a + (n-1)b)[/tex]

Dimana:

a = suku pertama = 4

b = selisih antar suku = 4

maka:

[tex]S_n = \frac{n}{2}(2(4)+ (n-1)(4))\\\\S_n = \frac{n}{2}(8+ 4n - 4)\\\\S_n = \frac{n}{2}(4n+4)[/tex]

maka jumlah 5 suku pertama adalah:

[tex]S_5 = \frac{5}{2}(4(5)+4)\\\\S_5 = \frac{5}{2}(24)\\\\S_5 = 60[/tex]

soal 2a: 1 + 3 + 9+ ...

rumus deret geometri:

[tex]S_n = \frac{a(r^n-1)}{r-1}[/tex]

dimana:

a = suku pertama = 1

r = rasio = 3

maka:

[tex]S_n = \frac{(1)(3^n-1)}{3-1}\\\\S_n = \frac{3^n-1}{2}\\\\S_7 = \frac{3^7-1}{2}\\\\S_7 = \frac{2187-1}{2}\\\\S_7 = 1093[/tex]

soal 2b: 32 + 16 + 8 + ...

rumus deret geometri untuk r < 1:

[tex]S_n = \frac{a(1-r^n)}{1-r}[/tex]

dimana:

a = 32

b = 1/2

maka:

[tex]S_n = \frac{32(\left1-(\frac{1}{2})^n\right)}{1-1/2}\\\\S_n = \frac{32(\left1-(\frac{1}{2})^n\right)}{1/2}\\\\[/tex]

[tex]S_n = 64(\left1-(\frac{1}{2})^n\right)\\\\S_7 = 64(\left1-(\frac{1}{2})^7\right)\\\\S_7 = 64(\left1-\frac{1}{128}\right)\\\\[/tex]

[tex]S_7 = 64\left(\frac{127}{128}\right)\\\\S_7 = \frac{127}{2}[/tex]

[tex]$\begin{align} {x_2}^2&=4x_1\\(2x_1-12)^2&=4x_1\\4{x_1}^2-48x_1+144&=4x_1\\4{x_1}^2-52x+144&=0\\{x_1}^2-13x_1+36&=0\\(x_1-4)(x_1-9)&=0\\x_1&=4\ $atau$\ 9 \end[/tex]

[tex]$\begin{align} x_2&=2x_1-12\\&=2\times4-12\\&=-4 \end[/tex]

Substitusikan nilai x₁ = 9

[tex]$\begin{align} x_2&=2\times9-12\\&=6 \end[/tex]

Maka, persamaan kuadratnya

[tex]$\begin{align} (x-9)(x-6)&=0\\x^2-15x+54&=0 \end[/tex]

Sehingga, nilai diskriminan persamaan kuadrat tersebut adalah

[tex]$\begin{align} D&=b^2-4ac\\&=(-15)^2-4\times1\times54\\&=225-216\\&=\boxed9 \end[/tex]
