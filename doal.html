6. contoh doal konstanta pegas


contoh soal :
1. Lima pegas masing-masing mempunyai konstanta 200 N/m disusun secara seri. Konstanta pegas pengganti dari susunan pegas tersebut adalah…

Pembahasan
Diketahui :
Konstanta masing-masing pegas (k1 = k2 = k3 = k4 = k5) = 200 N/m
Ditanya : konstanta pegas pengganti (k)
Jawab :

Konstanta pegas pengganti :
1/k = 1/k1 + 1/k2 + 1/k3 + 1/k4 + 1/k5
1/k = 1/200 + 1/200 + 1/200 + 1/200 + 1/200
1/k = 5/200
k = 200/5
k = 40 Newton/meter

29. Cara menyelesaikan doal 2sinx + sin3x = sin5x


jawab

2 sin x + sin 3x = sin 5x
2 sin x = sin 5x - sin 3x
2 sin x = 2 cos 1/2 (5x + 3x)  sin 1/2(5x-3x)
2 sin x = 2 cos 4x sin x
2 sin x - 2 cos 4x  sin x = 0
2 sin x (1 - cos 4x) = 0
2 sinx = 0  atau 1 - cos 4x = 0
sin x = 0  atau cos 4x = 1

untuk interval  0≤ x ≤  360
sin x = 0 = sin 0 = sin 360
x =  (0, 360)

cos 4x = 1 = cos 0
4x = 0 + k. 360
x = 0 + k. 90
x = (0, 90, 180,  270, 360)
