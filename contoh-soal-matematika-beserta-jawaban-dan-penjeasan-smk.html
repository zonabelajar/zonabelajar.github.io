7. pengertian dan contoh rumus fungsi matematika untuk smk?


Fungsi adalah suatu relasi khusus yang memasangkan setiap elemen2 daerah asal(domain) tepat satu elemen2 daerah kawan (kodomain) .  jadi agar suaturelasi dapat disebut fungsi harus memenuhi hal2 sbb : semua elemen daerah asal harus memiliki pasangan dan dan setiap pasangan yg ada adalah tepat satu.
contoh : diket himpunan A : {1,2,3,4,5} dan B : {3,5,7,9}. manakah dari relasi dibawah ini yg merupan fungsi 
R1 : { (1,3) (2,5) (3,7) (4,9) (5,3) }
R2 : { (1,3) (1,5) (2,7) (3,9) }
R3 : { (1,5) (2,5) (3,5) (4,5) (5,9) }
jawab :
R1, R3 merupakan fungsi
R2 bukan fungsi karena ada elemen A yang tidak mempunyai pasangan

20. tolong di bantu yah, soal matematika SMK kelas XI ​


a. Persamaan lingkaran berpusat (-2,5), melalui (-1,2)

(x + 2)² + (y - 5)² = r²

→ (-1 + 2)² + (2 - 5)² = 10

→ x² + y² + 4x - 10y + 4 + 25 - 10 = 0

→ x² + y² + 4x - 10y + 19 = 0

b. Persamaan lingkaran berpusat (-2,5) menyinggung garis x - 3 = 0 → x = 3

(x + 2)² + (y - 5)² = r²

→ (3 + 2)² + (0 - 5)² = r²

→ r² = 25

→ (x + 2)² + (y - 5)² = 25

→ x² + y² + 4x - 10y + 4 + 25 - 25 = 0

→ x² + y² + 4x - 10y + 4 = 0

26. Cara perhitungan Logaritma Dengan Kalkulator? Tolong di terangin caranya,dan kasih penjelasan beserta cara pakai kalkulator_nya. Ini contoh soalnya: ²log8 Tolong ya yang bisa di jawab yang benar... Saya akan kasih doa diberikan keselamatan jika ada yang benar menjawabnya. Terimakasih Matematika kelas 10 (SMK)


Jawaban:

Tentu, saya akan dengan senang hati menjelaskan cara menghitung logaritma menggunakan kalkulator dan memberikan contoh langkah-langkahnya. Pertama, mari kita perhatikan contoh soalnya: \(2\log_8\).

Langkah-langkahnya adalah sebagai berikut:

1. Nyalakan kalkulator Anda dan pastikan dalam mode "logaritma" atau "log".

2. Masukkan angka yang ingin Anda hitung logaritmanya, dalam hal ini, \(2\).

3. Tekan tombol "log" atau "logarithm".

4. Masukkan basis logaritma, dalam kasus ini, \(8\).

5. Tekan tombol "=" atau "enter" untuk mendapatkan hasil.

Contoh perhitungan:

\(2\log_8\)

1. Masukkan angka \(2\).

2. Tekan tombol "log", biasanya ada tombol yang bertuliskan "log" di kalkulator.

3. Masukkan basis logaritma, yaitu \(8\).

4. Tekan tombol "=" atau "enter".

Hasilnya akan muncul di layar kalkulator. Pastikan untuk melihat di layar apakah hasilnya sudah muncul.

Dalam contoh ini, hasil logaritma \(2\log_8\) akan muncul di layar kalkulator.

Semoga penjelasan ini membantu menghitung logaritma dengan kalkulator. Jika memiliki pertanyaan lebih lanjut, jangan ragu untuk bertanya. Saya menghargai doa dan harap yang saya berikan bermanfaat bagi Anda!

35. soal matematika kelas 11 smk, dimohon bantuan nya


Penjelasan dengan langkah-langkah:

1.

h(x) = f(g(x))

h(x) = f(2x-4)

h(x) = 5 + 2(2x - 4)

h(x) = 5 + 4x - 8

h(x) = 4x - 3

2.

h^(-1)(4x - 3) = x

misalkan

(4x - 3) = a

4x = a + 3

x = (a + 3)/4

Jadi,

h^(-1)(a) = (a + 3)/4

Ubah a jadi x,

h^(-1)(x) = (x + 3)/4

3.

k(x) = g(f(x))

k(x) = g(2x + 5)

k(x) = 2(2x + 5) - 4

k(x) = 4x + 10 - 4

k(x) = 4x + 6

4.

k^(-1)(4x + 6) = x

misalkan.

(4x + 6) = a

4x = a - 6

x = (a - 6)/4

Jadi,

k^(-1)(a) = (a - 6)/4

Ubah a jadi x,

k^(-1)(x) = (x - 6)/4

5. Pulau Dana

Pulau Dana adalah sebuah pulau kecil yang terletak di perairan sebelah barat daya Pulau Sawu. Secara administratif, pulau ini termasuk wilayah Kabupaten Sabu Raijua, Provinsi Nusa Tenggara Timur. Potensi perikanan dan panoramanya yang menawan, membuat pulau ini rawan okupasi Australia.
