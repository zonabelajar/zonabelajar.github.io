25. Carilah hasil kali cartesius (Cartesian product) dari himpunan-himpunan berikut: S1 = {a, b, c}; S2 = {1, 2, 3}; S3 = {d, e} a) S1 x S2 b) S1 x S3 c) S2 x S3 d) S3 x S1 e) S3 x S2 f) S1 x S2 x S3


Jawab:

Penjelasan dengan langkah-langkah:

Untuk mencari hasil kali Cartesius (Cartesian product) dari himpunan-himpunan yang diberikan, kita menggabungkan setiap elemen dari himpunan pertama dengan setiap elemen dari himpunan kedua, dan seterusnya. Berikut adalah hasil kali Cartesius dari himpunan-himpunan yang diberikan:

a) S1 x S2 = {(a, 1), (a, 2), (a, 3), (b, 1), (b, 2), (b, 3), (c, 1), (c, 2), (c, 3)}

b) S1 x S3 = {(a, d), (a, e), (b, d), (b, e), (c, d), (c, e)}

c) S2 x S3 = {(1, d), (1, e), (2, d), (2, e), (3, d), (3, e)}

d) S3 x S1 = {(d, a), (d, b), (d, c), (e, a), (e, b), (e, c)}

e) S3 x S2 = {(d, 1), (d, 2), (d, 3), (e, 1), (e, 2), (e, 3)}

f) S1 x S2 x S3 = {(a, 1, d), (a, 1, e), (a, 2, d), (a, 2, e), (a, 3, d), (a, 3, e),

(b, 1, d), (b, 1, e), (b, 2, d), (b, 2, e), (b, 3, d), (b, 3, e),

(c, 1, d), (c, 1, e), (c, 2, d), (c, 2, e), (c, 3, d), (c, 3, e)}

Hasil kali Cartesius merupakan himpunan baru yang terdiri dari semua kombinasi elemen-elemen dari himpunan-himpunan yang diberikan.
