10. plissss jawab dong aku binggungggg Sebuah file didowload di internet selama 3 menit kecepatan modem yang mendowload nya adalah 150 kbps berapa MB kah ukuran dari fike tersebut !!


asumsikan kecepatan download stabil.
150 kbps artinya adalah 150 kilobit per second.

Jika total waktu download adalah 3 menit, maka ukuran file adalah:
3 menit = 3*60 detik = 180 detik.
150 kilobitperdetik * 180 detik = 27.000 kilobit (Kb)

1 byte = 8 bit
27.000/8 = 3375 byte (B) = 3,375 megabyte (MB)

Maka, ukuran file adalah 3,375 MB (tiga koma tiga tujuh lima megabyte).
