6. jelaskan menu utama yang digunakan untuk membuat data bisnis baru pada aplikasi komputer akuntansi MYOB


Menu utama yang digunakan untuk membuat data bisnis baru pada aplikasi komputer akuntansi MYOB adalah Create A New Company File.

Pembahasan

MYOB (Mind Your Own Businnes) Accounting atau biasa dikenal sebagai salah satu aplikasi komputerisasi akuntansi yang sering digunakan oleh perusahaan untuk mengentri data keuangan agar menjadi lebih efektif. MYOB sangat berguna untuk merekam seluruh kegiatan transaksi perusahaan dibandingkan cara manual. Fungsi MYOB adalah memudahkan kinerja akuntan dalam membuat pembukuan hingga laporan keuangan perusahaan, menghindari kesalahan hitung yang terjadi karena sistem yang digunakan secara otomatis, dan fitur-fitur yang ada pada MYOB dapat mempraktisi pekerjaan akuntan perusahaan.

Berikut beberapa menu utama pada program MYOB yang biasa digunakan:

Open Your Company File adalah menu yang fungsinya untuk membuka data akuntansi perusahaan yang sudah pernah dibuat.Create A New Company File adalah menu yang digunakan untuk mengentri data akuntansi perusahaan yang baru.Explore The Sample Company adalah menu yang digunakan untuk mengeksplor contoh data akuntansi perusahaan yang ada di MYOB.What’s New in This Version adalah menu yang didalamnya terdapat informasi tambahan tentang fitur baru dan peningkatan dari versi sebelumnya.Exit MYOB Accounting adalah menu untuk keluar dari program.

Dalam membuat data bisnis baru maka menu yang dipilih adalah Create A New Company File, berikut langkah-langkah dalam membuat data baru :

Aktifkan program MYOB dengan cara klik tombol Start atau klik double pada ikon MYOB.Selanjutnya akan tampil jendela “Welcome To MYOB Accounting” yang berisikan Open Your Company File, Create A New Company File, Explore The Sample Company, What’s New in This Version, dan Exit Myob Accounting.Klik menu “Create A New Company File”, lalu klik next.Masukkan data perusahaan. Setelah itu klik next.Masukkan informasi akuntansi, dan klik next bila sudah selesai.Setelah melakukan konfirmasi bahwa tahun keuangan sudah benar, lalu klik next.Pilih cara untuk pembuatan daftar rekening, apakah akan memakai daftar rekening yang sudah disediakan MYOB, mengimpor dari tempat lain atau membuat sendiri. Kita pilih memakai daftar rekening dari MYOB (opsi pertama), lalu klik next.Isikan retail pada industry classification dan computer dealer pada business type. Kemudian klik Next.Atur folder dimana anda akan meletakkan file MYOB yang sedang Anda buat saat ini. Jika sudah klik next.Selanjutnya akan tampil informasi proses telah selesai. Lalu klik Setup Assistance, maka akan muncul jendela Easy Setup Assistant.

15. eksistensi file dari program software komputer akuntansi accurate adalah... a. gbk b. odp c. xls d. doc e. gdb


Eksistensi file dari program software komputer akuntansi accurate adalah e. GDB. Akuntansi accurate termasuk ke dalam salah satu program yang membantu praktik akuntansi dan keuangan.

Pembahasan

Jawaban e. GDB bebar dikarenakan kepanjangan dari GDB adalah  File Database InterBase. Sehingga eksistensi file tersebut tepat untuk software akuntansi yang mengolah data. Accurate Accounting Software adalah program akuntansi dan keuangan yang dirancang untuk memenuhi kebutuhan perusahaan dalam hal pencatatan akuntansi dan keuangan. Accurate telah dipakai lebih dari 10.000 perusahaan di Indonesia dan merupakan program akuntansi dan keuangan terkemuka di Indonesia. Berdiri pada 26 Oktober 1998, Cipta Piranti Sejahtera, yang lebih di kenal dengan nama CPSSoft, sebagai pengembangan software. Misi bisnis adalah menyediakan software yang berkualitas tinggi untuk usaha kecil dan menengah dengan harga yang terjangkau.

Software accurate yang sudah dipakar oleh lebih dari 40.000 perusahaan dengan kondisi bisnis yang berbeda-beda dan sangat mudah penggunaannya serta memberikan informasi yang real time proses. Accurate juga memiliki laporan yang lengkap dengan kombinasi lebih dari 250 laporan baik dalam bentuk grafik maupun persentasi untuk analisa. Tersedianya laporan keuangan (Balance Sheet, Income Statement, Cash Flow) yang bisa dibuat dengan perbandingan Tahun, Bulan, Laporan Penjualan, Laporan Piutang & Umur Piutang, Laporan Hutang & Umur Hutang, Laporan Pembelian, Laporan Persediaan Barang, Kartu Stock, Laporan Pajak dan Laporan-laporan pendukung lainnya.

20. Jelaskan menu utama yang digunakan dalam membuat data bisnis baru pada aplikasi komputer akuntansi MYOB​


Jawaban:

- Cara membuat file data bisnis baru menggunakan program MYOB Accounting, simak pada langkah-langkah berikut ini :

1.) Silahkan buka menu MYOB anda.

2.) Setelah menu Welcome To MYOB terbuka, selanjutnya silahkan klik Create A New Company File.

3.) Setelah anda klik Create A New Company File, selanjutnya anda akan dibawa ke tampilan Form New Company File Assistant Intoduction.

4.) Selanjutnya silahkan klik Next>

5.) Setelah itu anda akan masuk ke dalam menu Windows Company Information. Silahkan untuk melegkapi data pada kolom yang tampil di program MYOB anda. Pada formulir tersebut akan tampil Serial Number, Company Name, Company No, Vat No, Addres, Phone Number, Fax Number, Email Addres. Setelah selesai mengisi formulir tersebut silahkan klik Next>

6.) Selanjutnya anda akan dibawa pada halaman Windows Accounting Information. Disini kita juga harus melengkapi data seperti pada halaman menu Windows Company Profile. Pada laman ini silahkan isi data pada Current Funancial Year (periode tahun akuntansinya), Last Month of Financial Year (bulan terakhir periode akuntansi sebelumnya), Conversion Month (bulan yang akan kita proses atau di bukukan) dan Number of Periods (silahkan pilih twelve atau thirteen jika data penyesuaian dicatat khusus). Setelah melengkapi semua data diatas selanjutnya silahkan untuk klik Next>

7.) Sebelum lanjut ke langkah selanjutnya, sebaiknya periksa kembali data yang telah anda entry. Bila ada kesalahan dalam entry data silahkan untuk klik <Back untuk memperbaharuinya. Apabila telah selesai silahkan klik Next>

8.) Setelah itu pilihlah akun pada menu Window Account List sesuai dengan kebutuhan anda.

9.) Setelah itu, pada Window Company File silahkan untuk menentukan directory dan folder serta nama file yang akan digunakan tersimpan. File ini secara default akan tersimpan pada C:\Accounting 17\nama perusahaan. Jika anda ingin menggantinya sesuai dengan kebutuhan silahkan untuk klik tombol Change lalu tentukan Directory dan Folder serta nama file yang baru. Setelah semuanya selesai silahkan untuk klik >Next

10.) Jika anda sudah sampai pada Window Conclusion, berarti tanda pembuatan file data bisnis baru sudah selesai. Selanjutnya silahkan untuk klik tombol Command Centre untuk masuk ke menu utama MYOB Accounting. Selanjutnya akan tampil Command Centre MYOB dengan tampilan sesuai file data yang telah anda buat sebelumnya.

23. Apa aplikasi yang digunakan oleh file dengan extension doc


Aplikasi yang digunakan oleh fail dengan ekstensi doc adalah aplikasi pengolah kata. Aplikasi-aplikasi ini misalnya adalah Google Docs dan Microsoft Office Word.

Pembahasan

Word processor, atau dalam bahasa Indonesia disebut "pengolah kata", adalah jenis aplikasi atau program komputer yang menyediakan layanan pengolahan kata. Aplikasi jenis ini sangat penting digunakan dalam membuat beragam macam teks atau wacana, seperti dokumen, surat, catatan, undangan, formulir dan sebagainya.

Aplikasi pengolah kata umumnya menyediakan layanan membuat dokumen, memformat dokumen, menyunting/mengedit dokumen dan mencetak dokumen. Google Docs dan MS Word adalah dua jenis aplikasi pengolah kata yang saat ini paling populer digunakan. Google Docs sendiri adalah layanan pengolah kata daring (online) sehingga untuk mengaksesnya, kita membutuhkan koneksi internet. MS Word di sisi lain dapat tersedia secara luring (offline)
