Tool Box

Tool Box adalah suatu kotak atau alat yang memiliki fungsi untuk menggambar, modifikasi objek, memberi warna, menulis teks. Tool Box memiliki beberapa fungsi di satu kotak atau tempat yang sama, jika ingin membuka atau mempergunakan fungi lain maka silahkan klik tanda segitiga kecil di pojok kanan bawah salah satu tool box.

Tool Bar

Tool Bar merupakan sekumpulan tombol shortcut yang memiliki fungsi untuk menjalankan perintah dengan cepat. ShortCut Ini adalah bagian yang paling sering digunakan oleh desainer grafis dalam melakukan sesuatu. CorelDraw memberikan fitur dalam toolbar yang powerfull namun mudah sekali digunakan.

Menu Bar

Menu Bar berisi tombol-tombol text yang memiliki banyak sekali fungsi yang berbeda sesuai dengan namanya masing-masing. Desainer pun sering menggunakan fasilitas ini biasanya dalam memberikan suatu effect berhubungan dengan vector dan bitmap.

Property Bar

Property Bar berisi sekumpulan tombol yang isinya itu menyesuaikan dengan tools yang sedang kita pilih di dalam tool box.

Docker

Docker merupakan kotak dialog yang di dalamnya berisi berbagai tombol perintah untuk mengatur, memberi effect kepada sebuah objek maupun teks.

Rulers

Rulers merupakan alat ukur berupa penggaris dibagian atas dari drawing page, dan di bagian kanan drawing page, rullers digunakan sebagai acuan yang mampu menunjukkan ukuran suatu objek atau dokumen, serta tata letak suatu desain.

Status Bar

Status Bar terletak dibagian bagian bawah area program sebelah kanan. Status Bar berisi informasi mengenai objek (teks) seperti ukuran, bentuk, warna, outline, format gambar, jenis huruf beserta  ukurannya.

Drawing Page

Drawing Page adalah adalah sebuah area kerja utama di dalam CorelDraw di mana Anda dapat mendesain suatu objek di dalamnya dan merupakan area cetak suatu desain.

Color Palette

Color Palette bagian dari CorelDraw yang berisi koleksi warna di dalam sebuah bar dan secara default/bawaan terletak di sebelah kanan dokumen / program.

15. macam-macam garis pada coreldraw​


Jawaban:

Pick Tool adalah tools untuk seleksi dan mengubah ukuran objek pada lembar kerja CorelDRAW

Shape Tool adalah tool untuk mengubah bentuk objek pada lembar kerja CorelDRAW. Dalam tool ini, tersimpan 4 jenis tool berbeda, yang fungsinya saling berkaitan, yakni : Shape, Smudge Brush, Roughen Brush, dan Free Transform. Adapun tombol shortcut atau jalan pintas yang dapat digunakan untuk mengakses tool ini secara cepat adalah tombol F10.

Crop Tool adalah tool yang dapat digunakan untuk memotong sebagian dari objek pada lembar kerja CorelDRAW yang tidak di inginkan. Dalam tool ini, tersimpan 4 jenis tool berbeda, yang fungsinya saling berkaitan, yakni : Crop, Knife, Eraser, dan Virtual Segment Delete. Tool ini tidak memiliki tombol shortcut.

Zoom Tool adalah tool yang digunakan untuk memperbesar ataupun memperkecil tampilan objek pada lembar kerja CorelDRAW. Dalam tool ini, tersimpan 2 jenis tool berbeda, yang fungsinya saling berkaitan, yakni : Zoom (shortcut Z) dan Pan (shortcut H).

Freehand Tool adalah tool yang dapat digunakan untuk membuat berbagain macam garis dalam lembar kerja CorelDraw, baik garis bebas, point to point, maupun garis-garis yang lain. Terdapat banyak sekali tool yang tersimpan didalam tool freehand yang fungsinya saling berkaitan satu sama lain. Tool-tool tersebut adalah Freehand, 2-Point Line, Bézier, Artistic Media, Pen, B-Spline, Polyline, serta 3-Point Curve. Tombol yang dapat digunakan untuk mengakses tool Freeand secara cepat/shortcut tombolnya adalah tombol F5, sedangkan untuk mengakses tool Artistic Media, kita dapat menggunakan tombol I.

Perbedaan antara cara menggabungkan beberapa file melalui Embed dan Copy-Paste adalah bahwa pada Embed, file yang digabungkan menjadi satu file, sementara pada Copy-Paste, file yang digabungkan tetap terpisah.

Keunggulan menempatkan objek dengan sistem Link adalah dapat mengurangi ukuran file karena objek tidak disimpan dalam file utama, dan juga memudahkan proses update karena perubahan pada objek akan terupdate secara otomatis pada file utama.

Langkah-langkah untuk menyimpan gambar dari CorelDraw dalam format wmf adalah: Pilih menu File > Save As, pilih format WMF, atur nama file dan lokasi penyimpanan, kemudian klik Save.

Format file dari CorelDraw yang dapat dikenal oleh aplikasi Microsoft Word adalah format JPG, PNG, atau BMP.

Teks pada Microsoft Word dapat disalin ke CorelDraw dengan cara memilih teks yang ingin dicopy di Microsoft Word, kemudian klik kanan dan pilih Copy atau tekan tombol Ctrl+C, kemudian buka CorelDraw dan klik kanan pada dokumen kosong, lalu pilih Paste atau tekan tombol Ctrl+V.

Perbedaan antara Link dan Embed adalah bahwa pada Link, objek yang dimasukkan ke dalam dokumen hanya berupa tautan yang mengarah pada file terpisah, sementara pada Embed, objek yang dimasukkan menjadi bagian dari dokumen utama.

Fungsi ikon paste adalah untuk memasukkan isi yang telah disalin atau dipotong ke dalam dokumen yang sedang dibuka.

Fungsi perintah Insert pada Microsoft Word adalah untuk menyisipkan berbagai macam elemen ke dalam dokumen seperti gambar, tabel, diagram, hyperlink, dan lain-lain.

Fasilitas Text Wrapping pada Microsoft Word digunakan untuk mengatur letak objek dalam dokumen, sehingga objek dapat ditempatkan di antara teks atau di atas teks.

Ikon Copy pada aplikasi CorelDraw terdapat di toolbar standar, atau dapat juga dilakukan dengan menekan tombol Ctrl+C.
