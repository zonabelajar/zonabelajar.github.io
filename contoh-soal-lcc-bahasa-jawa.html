Nilai a dan b yang mungkin adalah:

a = 1, b = 63,a = 3, b = 21,a = 7, b = 9.a = 9, b = 7,a = 21, b = 3, dana = 63, b = 1

Atau HP = { (a, b) | (1, 63), (3, 21), (7, 9), (9, 7), (21, 3), (63, 1) }

 

Pembahasan

Jika x adalah 3, f adalah 5, dan x·f·a·b = 945, maka dengan cara faktorisasi prima dan dengan asumsi bahwa a, b ∈ ℕ (a dan b adalah bilangan asli), dapat diperoleh:
3 · 5 · a · b = 945
⇒ 3 · 5 · a · b = 3 · 315
⇒ 3 · 5 · a · b = 3 · 3 · 105
⇒ 3 · 5 · a · b = 3 · 3 · 3 · 35
⇒ 3 · 5 · a · b = 3 · 3 · 3 · 5 · 7
⇒ 3 · 5 · a · b = 3 · 5 · 3 · 3 · 7
⇒ a · b = 3 · 3 · 7
Jangan lupa bahwa  a · b juga bisa bernilai 1 · 63 atau 63 · 1.

 

KESIMPULAN

Nilai a dan b yang mungkin adalah:

a = 1, b = 63,a = 3, b = 21,a = 7, b = 9.a = 9, b = 7,a = 21, b = 3, dana = 63, b = 1

Dalam bentuk himpunan penyelesaian, solusi di atas dapat dinyatakan dengan:
HP = { (a, b) | (1, 63), (3, 21), (7, 9), (9, 7), (21, 3), (63, 1) }
