7. tolong bantu jawab bahasa Inggris yaa teman teman kunci jawabannya antara TRUE or FALSE


Answers:

1. FALSE ⇒ There isn't any/There's no muscle in the body stronger than the tongue.

2. TRUE  

3. FALSE ⇒ Scarcely any train stations are larger than Grand Central Station in New York City.

4. FALSE ⇒ I have nothing/don't have anything in my purse, so I don't care if it's stolen.

5. FALSE ⇒Jerry never visited any place he liked better than home, so he finally started staying home most of the time.

6. TRUE  

7. TRUE

8. FALSE ⇒We will never accept any new production standards.

9. FALSE ⇒Many cats have hardly any tails.

10. FALSE ⇒He could not lend me $5 because he had only $3.

11. TRUE

12. FALSE ⇒The rules required us to form an orderly line and not to talk.

13. TRUE

14. FALSE ⇒You are not going to finish the test in time.

15. TRUE

----------

Explanation

No yang berarti "not any" diikuti kata benda atau comparative adjective (e.g. no sooner than, no longer).

Kata no bermakna negatif.

1. ada kata isn't = is not; jadi tidak perlu kata no karena not any = no

3. Scarcely = jarang; maknanya negatif, jadi jangan gunakan no lagi. Selalu hindari double negative.

4. don't = do not; nothing = not anything

5. never = tidak pernah; maknanya negatif. Lihat penjelasan untuk no 3

8. Lihat no 5

9. hardly = hampir tidak

10. only = hanya; bermakna negatif

11. none of + (pro)noun

12. Setelah to gunakan infinitive; tidak perlu kata kerja bantu do

14. going bukan kata benda, jadi jangan gunakan no tapi not

15. Rarely = amat jarang; sudah negatif, tidak perlu no.

-----------

Details

Subject: English

Category: Structure and Written Expressions

Level: SHS

Code: 5

Keywords: no, none, not, negative

D. TRUE, TRUE, FALSE

Untuk menyelesaikan masalah ini, kita perlu menerapkan aturan logika Boolean, di mana NOT mengambil prioritas tertinggi, kemudian OR, dan terakhir AND. 10. Kalimat belum mempunyai kata kerja utama. Gunakan kata kerja positive karena sudah ada no.

35. Kode berikut ini yang akan menghasilkan nilai true adalah....a. if (true != false && false == false) b. if (false != false II true != true)c. if (true == true && false != false)d. if (false == false && true == false)​


Jawaban:

Jawaban yang benar adalah:

c. if (true == true && false != false)

Penjelasan:

Penjelasan:

Pada a, kondisi pertama (true != false) benar, namun kondisi kedua (false == false) juga benar. Karena ada operator AND (&&) maka hasilnya false.

Pada b, kondisi pertama dan kedua sama-sama salah (false != false dan true != true). Karena ada operator OR (||) maka hasilnya juga false.

Pada c, kondisi pertama (true == true) benar dan kondisi kedua (false != false) juga benar. Karena ada operator AND (&&) maka hasilnya true.

Pada d, kondisi pertama (false == false) benar, namun kondisi kedua (true == false) salah. Karena ada operator AND (&&) maka hasilnya false.

36. jawab plis di kumpulin hari ini :)kasik penjelasan dalam bahasa inggris mengapa kok true dan false ??​


Jawaban:

1. The rooster is shorter than the goat.

Artinya:

Ayam jantan itu lebih pendek dari kambing itu.

Answer:

T(True),karena faktanya kambing lebih besar postur badannya dari ayam jantan.

2. The fish is faster than the goat.

Artinya:

Ikannya lebih cepat dari kambing.

Answer:

T(True),karena faktanya jika ibaratnya ikan di berikan kolam yang besar, kemudian di sebelah kolamnya ada kambing yang luas tempatnya sama dengan kolam tersebut dan jika di kedua hewan itu di kejutkan maka otomatis yang memiliki reflek gerak tercepat saat terkejut adalah ikan dari pada kambing.

3. The tiger is the oldest animal.

Artinya:

Harimau adalah binatang yang paling tertua.

Answer:

F(False),riset mengatakan bahwa ada beberapa hewan yang memiliki usia tertua di bumi seperti: buaya, komodo, mamut dll.

4. The goat is younger than the hippopotamus.

Artinya:

Kambing itu paling muda dari kuda Nil.

Answer:

T(True),karena kita sudah bisa mengira bahwa kuda Nil memiliki usia yang agak tua dari kambing dan yang dapat terpikirkan oleh saya adalah jika kuda Nil menjadi musuh dari buaya maka kedua hewan tersebut sudah berusia sangat tua sekali.

5. The hippopotamus is tallest among the animals.

Artinya:

Kuda Nil adalah hewan yang tertinggi dari semua binatang.

Answer:
