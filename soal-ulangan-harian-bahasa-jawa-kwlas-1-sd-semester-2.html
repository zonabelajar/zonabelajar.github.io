17. Suatu pesawat udara mempunyai tempat duduk tidak lebih dari 50 penumpang. Setiap hari penumpang kelas utama boleh membawa bagasi 70kg, sedangkan untuk kelas ekonomi 30kg. Berapa keuntungan maksimum diperoleh maskapal penerbangan itu?


Jawaban:

x + y ≤ 50

70x + 30y ≤ 2100 (bagi 10)

⇔ x + y ≤ 50 (x7)

7x + 3y ≤ 210 (x1)

⇔4y ≤ 140

⇔y ≤ 35

x + y ≤ 50

x = 50 - 35 = 15

kelas utama (x) : 15 x Rp.250.000 = Rp.3.750.000

kelas ekonomi (y): 35 x Rp.175.000 = Rp.6.125.000

keuntungan maksimum: Rp.3.750.000 + Rp.6.125.000 = Rp.9.875.000.
