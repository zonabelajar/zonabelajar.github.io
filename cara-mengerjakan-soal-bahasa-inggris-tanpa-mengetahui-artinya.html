To answer the question which you didn't know what is the meaning, you can use intelligent guessing. For example we will translate ENLARGE.

• En- is some preposition to state some work. Example enchanted is "mempesona", so we can guess that is mem-.
• Large in dictionary mean besar

So, the meaning of enlarge is "memperbesar".
