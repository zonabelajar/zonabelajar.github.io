Tan 120 derajat. 2.dengan menggunakan rumus perbandingan (180 derajat + a derajat ) cos 240 derajat. 3.dengan menggunakan rumus perbandingan (270 derajat + a derajat ) cos 300 derajat.


1. utk sudut penyiku (90&270) sin menjadi cos dan cos menjadi sin
120= 90+30, sehingga sin120=cos30=1/2√3 dan cos120=sin30=1/2
tan=sin/cos => tan120=sin120/cos120= 1/2 √3 / 1/2=√3
krn 120 merupakan kuadran dua maka nilai tan & cos negatif, jadi tan120= -√3
2. untuk sudut pelurus (180&360) cos tetap cos, sin tetap sin
240=180+60
cos240=cos60=1/2
karena 240 berada di kuadran 3 maka nilai cos dan sin negatif. jadi cos240= -1/2
3. 300=270+30
cos300=sin60=1/2
karena 300 di kuadran 4 maka hanya cos yg positif, jadi nilai cos300=1/2

14. buktikan identitas Trigeometri berikut. Cos x / 1+ Sin x - 1+ Sin x / Cos x


Diketahui :

Cos x / 1+ Sin x - 1+ Sin x / Cos x = ... ?

Ditanya :

Pembuktian ... ?

Jawab :

Cos x / 1+ Sin x - 1+ Sin x / Cos x = - 2tan x

--------------------

PEMBAHASAN :

[tex]=\frac{cosx}{1+sinx} - \frac{1+sinx}{cosx}\\=\frac{(cosx)(cosx)}{(1+sinx)(cosx)} - \frac{(1+sinx)(1+sinx)}{(cosx)(1+sinx)}\\= \frac{cos^2x-1-2sinx-sin^2x}{(cosx)(1+sinx)}\\=\frac{cos^2x-(sin^2x+cos^2x)-2sinx-sin^2x}{(cosx)(1+sinx)}\\=\frac{-2sinx-2sin^2x}{(cosx)(1+sinx)}\\= \frac{-2sinx(1+sinx)}{(cosx)(1+sinx)}\\= -\frac{2sinx}{cosx}\\= - 2tanx[/tex]

Jadi, terbukti bahwa Cos x / 1+ Sin x - 1+ Sin x / Cos x adalah -2tan x

--------------------

19. tentukan nilai bentuk trigeometri berikutsin 570 + cos 480-tan 660 derajat​


Menentukan hasil rasio trigonometri dari (360° + θ) dan (n ∙ 360° + θ).

Jika n adalah bilangan bulat positif maka rasio trigonometri dari (n ∙ 360° + θ) sama dengan rasio trigonometri dari (+ θ).

Oleh karena itu,

sin (n ∙ 360° +  θ) = sin θ;

cos (n  ∙ 360° +  θ) = cos θ;

tan (n  ∙ 360° +  θ) = tan θ;

csc (n ∙ 360° +  θ) = csc θ;

sec (n  ∙ 360° +  θ) = sec θ;

cot (n  ∙ 360° +  θ) = cot θ.

Rasio trigonometri dari (180° - θ) hasilnya sama dengan

sin (180° -  θ) = sin θ;

cos (180° -  θ) = - cos θ;

tan (180° -  θ) = - tan θ;

csc (180° -  θ) = csc θ;

sec (180° -  θ) = - sec θ;

cot (180° -  θ) = - cot θ.

Pembahasan

sin 570° + cos 480° - tan 660°

= sin (2. 360° + (-150°)) + cos (360° + 120°) - tan (2. 360° + (- 60°))

= sin (-150°) + cos 120° - tan (- 60°)

= sin -(180° - 30°) + cos (180° - 60°) + tan 60°

= - sin 30° - cos 60° + tan 60°

= [tex]- \frac {1} {2} - \frac {1} {2} + \sqrt {3} [/tex]

= [tex]- 1 + \sqrt {3} [/tex]

26. Rumus Trigeometri silahkan bantu genk


Penjelasan dengan langkah-langkah:

[tex]r=4\text{ m}[/tex]

[tex]\theta=30^o[/tex]

Jarak kaki tangga dengan tembok

[tex]\sin \theta=\frac{x}{r}[/tex]

[tex]x=r\sin \theta[/tex]

[tex]x=4\sin 30^o[/tex]

[tex]x=4\times\frac{1}{2}[/tex]

[tex]x=2\text{ m}[/tex]

Tinggi tembok (dari ujung tangga ke lantai)

[tex]\cos \theta=\frac{y}{r}[/tex]

[tex]y=r\cos \theta[/tex]

[tex]x=4\cos 30^o[/tex]

[tex]x=4\times\frac{1}{2}\sqrt{3}[/tex]

[tex]x=2\sqrt{3}\text{ m}[/tex]

28. buktikan identitas trigeometri berikut.plisss bantu

(sin x.(1+sinx) + cos² x) / (cos x(1 + sin x)) =
(sin x + sin² x + cos² x) / (cos x(1+sin x)) =
(sin x + 1) /(cos x(1+sin x)) = 1/cos x = sec x
