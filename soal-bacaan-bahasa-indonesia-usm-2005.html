9. tuliskan 20 Desember 2005 dan 3 Juli 2005dalam bahasa Jepang hiragana !​


Cara penyebutan bulan dalam bahasa jepang :

Biasanya bulan disebut "がつ" atau gatsu. Jadi penyebutan bulan bulan sepanjang tahun "bilangan bulan + gatsu (がつ)"

20 desember 2005

- 20 jika diterjemahkan ke dalam bahasa jepang akan disebut "二十" (ni-juu)

- seperti rumus tadi "bilangan bulan + gatsu" bilangan bulan desember adalah 12 yang kalau di terjemahan ke bahasa jepang menjadi "じゅうに"

(ju-ni) jadi, "じゅうに + がつ" menjadi "じゅうにがつ" (ji-yuunigatsu)

-2005 jika diterjemahkan ke dalam bahasa jepang akan disebut "にせんご"(nisen go)

penulisan tanggal di jepang tidak seperti penulisan tanggal di Indonesia

Jika di Indonesia itu 20 desember 2005,

Kalau jepang itu 2005, desember, 20

Jadi hiragananya adalah :

- にせんご, じゅうにがつ, 二十

3 juli 2005

- 3 jika diterjemahkan ke dalam bahasa jepang akan disebut "三" (san)

- juli, juli adalah bulan ke 7,dan 7 disebut "なな". Jadi "なな+がつ" menjadi "なながつ" (nanagatsu)

Atau bisa disebut juga "しちがつ"(shichigatsu)

-2005 jika diterjemahkan ke dalam bahasa jepang akan disebut "にせんご" (nisen go)

Jadi penulisan nya adalah :

- にせんご, なながつ, 三

Atau :

-にせんご, しちがつ, 三

Semoga membantu

20 Desember 2005

= にせんごねんじゅうにがつはつか

= Ni sen go-nen juu ni-gatsu hatsuka

3 Juli 2005

= にせんごねんしちがつみっか

= Ni sen go-nen shichi-gatsu mikka

PENJELASAN :

Penyebutan tanggal, bulan, tahun dalam bahasa Jepang dimulai dari tahun, bulan, baru tanggal. Contoh :

Indonesia = 20 Desember 2005

Jepang = 2005年 12月 20日

年 = Tahun

月 = Bulan

日 = Tanggal / hari

17. Tolong dibantu kak.. Usm stis


∫dx/(e^x + 1)

misal, e^x + 1 = u
e^x = u - 1
e^x dx = du
dx = du/(e^x)
dx = du/(u - 1)

∫dx/(e^x + 1) = ∫(du/(u - 1))/u
= ∫du/(u(u - 1))
= ln|u - 1| - ln|u| + C
= ln |(u - 1)/u| + C
= ln|e^x/(e^x + 1)| + C
= ln|(e^x + 1)/(e^x)|^(-1) + C
= -ln|(e^x + 1)/(e^x)| + C
= -ln|1 + 1/e^x| + C
= -ln|1 + e^(-x)| + C

35. bahasa mandarin dari 17 juli 2005


Hanzi :

二零零五年七月十七日。

Atau :

2005年 7月 17日

Pembahasan

Pola penanggalan bahasa Mandarin begitu unik, karena berbalikan dengan pola penanggalan bahasa Indonesia. Apabila di Indonesia, didahulukan tanggal, bulan dan tahun. Sementara di Mandarin didahulukan tahun, bulan dan tanggal. Pola Penanggalan :

Apabila di Mandarin:

Tahun - Bulan - Tanggal

二零零五年 - 七月 - 十七日。

二零零五年 : Tahun 2005.七月 : Bulan ke-7 (Juli).十七日 : Tanggal 17.
