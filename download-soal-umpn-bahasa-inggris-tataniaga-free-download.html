28. SOAL UMPN KIMIA POLBAN (UMPN 2019)


Jawaban:

122,5gram

Penjelasan:

pertama buat mula2, reaksi/terurai, sisa

kemudian cari mol KClO3 dan ubah ke gram

Jawaban:

(1) B. 122,5 gram

(2) B. +11,9 kkal

Penjelasan:

(1) [tex]\boxed{\star\ Persamaan\ Reaksi\ \star}[/tex]

[tex]KClO_{3}_{(aq)}+6\ HCl_{(aq)} \rightarrow KCl_{(aq)}+3\ H_{2}O_{(l)}+3\ Cl_{2}_{(g)}[/tex]

◈ Diketahui massa KCl yang terbentuk adalah sebanyak 74,5 gram.

◈ Massa molekul relatif KCl = 39 + 35,5 = 74,5 gram/mol

◈ Mol KCl = [tex]\frac{m}{Mr}[/tex] = [tex]\frac{74,5\ gram}{74,5\ gram/mol}[/tex] = 1 mol

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

[tex]\boxed{\boxed{Untuk\ mencari\ mol\ KClO_{3}, gunakan\ perbandingan\ koefisien\ pada\ reaksi.}}[/tex]

Mol KClO₃ = [tex]\frac{Koefisien\ KClO_3}{Koefisien\ KCl}\times mol\ KCl[/tex]

Mol KClO₃ = [tex]\frac{1}{1}\times 1\ mol[/tex]

Mol KClO₃ = [tex]1\ mol[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Mencari massa KClO₃ yang digunakan: (Mr KClO₃ = 122,5 gram/mol)

Mol KClO₃ = 1 mol

Mol KClO₃ = [tex]\frac{m}{Mr}[/tex]

Massa KClO₃ = [tex]mol \times Mr[/tex]

Massa KClO₃ = [tex]1\ mol \times 122,5\ gram/mol[/tex]

[tex]\boxed{Massa\ KClO_3 = 122,5\ gram}[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Jadi, massa kalium klorat yang diperlukan untuk menghasilkan 74,5 gram kalium klorida sebanyak dalam reaksi ini adalah sebanyak 122,5 gram.

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

(2) Diketahui perubahan entalpi untuk reaksi:

[tex]H_2O_{(l)} \rightarrow H_2_{(g)}+\frac{1}{2}\ O_2_{(g)}\ \ \Delta H = +68,3\ kkal\\H_2_{(g)}+\frac{1}{2}\ O_2_{(g)}\rightarrow H_2O_{(g)}\ \ \Delta H = -57,8\ kkal\\H_2O_{(l)} \rightarrow H_2O_{(s)}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \Delta H = -1,4\ kkal[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Perubahan entalpi dari es menjadi uap air:

[tex]H_2O_{(l)} \rightarrow H_2_{(g)}+\frac{1}{2}\ O_2_{(g)}\ \ \Delta H = +68,3\ kkal\\H_2_{(g)}+\frac{1}{2}\ O_2_{(g)}\rightarrow H_2O_{(g)}\ \ \Delta H = -57,8\ kkal\\H_2O_{(s)} \rightarrow H_2O_{(l)}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \Delta H = +1,4\ kkal[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Sehingga,

[tex]H_2O_{(s)} \rightarrow H_2O_{(g)} \ \ \ \ \ \ \ \ \Delta H =(68,3-57,8+1,4)kkal[/tex]

[tex]H_2O_{(s)} \rightarrow H_2O_{(g)} \ \ \ \ \ \ \ \ \Delta H = +11,9\ kkal[/tex]

[tex]\bigstar\ \bigstar\ \bigstar[/tex]

Jadi, perubahan entalpi yang terjadi untuk mengubah es menjadi uap air adalah sebesar + 11,9 kkal.
