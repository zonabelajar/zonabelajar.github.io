8. Soal-saol 1. (4xy+3y²-x)dx+x (x+2y)dy = 0sertakan caranya ​


Mencari faktor integrasi, yaitu suatu fungsi u(x,y) sehingga persamaan dapat dituliskan sebagai:

u(x,y) [ (4xy+3y²-x)dx+x (x+2y)dy ] = 0

Untuk mencari faktor integrasi, kita perlu memeriksa apakah persamaan ini memenuhi syarat persamaan yang dapat dituliskan dalam bentuk:

Mdx + Ndy = 0

Jika memenuhi, maka faktor integrasi dapat ditemukan dengan rumus:

u(x,y) = e^(∫(N/M)dy)

Jika tidak memenuhi, maka kita perlu melakukan transformasi terlebih dahulu.

Dalam hal ini, kita dapat melihat bahwa persamaan dapat dituliskan sebagai:

(4xy+3y²-x)dx = -x (x+2y)dy

Maka, kita perlu mencari faktor integrasi dengan rumus:

u(x,y) = e^(∫(N/M)dy) = e^(∫(-x(x+2y))/(4xy+3y²-x)dx)

Untuk mencari integral di atas, kita perlu mencari faktor pembagi terlebih dahulu. Kita dapat membagi kedua ruas persamaan dengan x(4y-1), sehingga diperoleh:

[(4y-1)/(4xy+3y²-x)]dx - dy/x = 0

Maka, kita dapat menggunakan faktor integrasi dengan rumus:

u(x,y) = e^(∫(-1/x)dx) e^(∫(1/(4y-1))dy)

u(x,y) = e^(-ln|x|) e^(ln|4y-1|)

u(x,y) = |4y-1|/|x|

Setelah kita menemukan faktor integrasi, kita dapat mengalikan persamaan awal dengan faktor integrasi tersebut, sehingga diperoleh:

|4y-1|dx - x(x+2y)dy/|x| = 0

Persamaan ini dapat diselesaikan dengan metode pemisahan variabel, yaitu dengan memindahkan semua variabel x ke satu sisi dan semua variabel y ke sisi lainnya, sehingga diperoleh:

|4y-1|dx = x(x+2y)dy/|x|

Kita dapat memisahkan tanda absolut dengan membagi persamaan menjadi dua kasus, yaitu:

4y-1 > 0 dan x > 0:

(4y-1)dx = x(x+2y)dy

4ydx + (1-x)dx = xdy + 2xydy

(4y-2xy)dy = (x-1)dx

(2y-x)dy + ydy = (x-1)dx/2

(2y-x+2y²/2)dy = (x-1)dx/2

y²-x+2y³/3 = x²/4 - x/2 + C

4y-1 > 0 dan x < 0:

-(4y-1)dx = x(x+2y)dy

-4ydx - (1+x)dx = xdy + 2xydy

(-4y+2xy)dy = (-x-1)dx

(2y-x)dy - ydy = (1+x)dx/2

(2y-x+2y²/2)dy = -(x+1)dx/2

y²-x+2y³/3 = -x²/4 - x/2 + C

Maka, solusi umum persamaan diferensial ini adalah:

y²-x+2y³/3 = x²/4 - x/2 + C

untuk 4y-1 > 0 dan x > 0

atau

y²-x+2y³/3 = -x²/4 - x/2 + C

untuk 4y-1 > 0 dan x < 0.

SISTEM OPERASI

Sistem operasi (bahasa Inggris: operating system; disingkat OS) adalah perangkat lunak sistem yang mengatur sumber daya dari perangkat keras dan perangkat lunak, serta sebagai jurik (daemon) untuk program komputer. -Managemen Sistem I/O.

19. contoh soal yang mencari f(x) suku banyak beserta jawabannya


jika diketahui f(x) = -4[tex] x^{3} [/tex] + 2[tex] x^{2} [/tex]  - 7x + 6 dan g(x) = 2[tex] x^{3} [/tex] - [tex] x^{2} [/tex] + 5x - 5 , tentukan f(x)+ g(x)?
jawab
f(x)+g(x)
=(-4[tex] x^{3} [/tex] + 2[tex] x^{2} [/tex] - 7x + 6) + (2[tex] x^{3} [/tex] - [tex] x^{2} [/tex] + 5x -5)
= -2[tex] x^{3} [/tex] + [tex] x^{2} [/tex] - 2x + 1
Diketahui suku banyak f(X) jika dibagi ( x+ 1) sisanya 8 dan ( x-3 ) sisanya 4 suku banyak g(x) jika dibagi dengan (x+1) bersisa -9 dan jika dibagi (x-3) sisanya 15 jika h(x) = F(X) . G(X), maka sisa pembagian H(X) oleh ×² - 2× - 3

Jawaban


H(x) = f(x) . g(x)
f(x) dibagi (x + 1) sisa  8                dbagi (x - 3) sisa 4
g(x) dibagi (x + 1) sisa -9                dibagi (x - 3) sisa 15

Fungsi

Fungsi adalah materi dalam Matematika. Fungsi dapat kita pelajari dari kelas 8. Fungsi sangat membantu kehidupan dalam sehari-hari. Fungsi yang kita pelajari ini, BUKAN yang dimaksud 'Guna/Manfaat' dari Kehidupan sehari- hari, Melainkan sebuah "Relasi" atau yang disebut "Aturan yang menghubungkan".

Apa sih yang disebut dengan Fungsi?

Fungsi adalah 'Relasi' khusus yang memasangkan setiap anggota himpunan A (domain) dengan tepat pada satu anggota himpunan B (kodomain). Fungsi juga dapat dikatakan sebagai 'Pemetaan' khusus. Mari, Saya buat soal seperti berikut ini.

Diket:

Himpunan A = 2,3,5

Himpunan B = 4,5,6,10    

Aturan yang menghubungkan/ Relasi yang tepat adalah " Dua kali dari "

Sehingga, kita dapat mengetahui Range dari Himpunan tersebut.

Maka, dari keterangan tersebut, dapat kita ketahui bahwa:

- Domain = Himp. A => { 2,3,5 }

- Kodomain = Himp. B => { 4,5,6,10 }

- Range (Daerah Hasil) => {4,6,10} atau >> ( {2,4} {3,6} {5,10} )

Banyak jenis dari "Fungsi" seperti Korespondesi 1-1, Komposisi.
