11).

Mencari nilai x dari :

[tex]3x + 4 = 2(x - 4) \\ 3x + 4 = 2x - 8 \\ 3x - 2x = - 8 - 4 \\ x = - 12[/tex]

diperoleh nilai x = -12 kita subsitusikan ke :

= x + 6

= -12 + 6

= -6

Jawaban (C) ✅

12).

Diketahui :

panjang (3x - 6) meter

lebar (x + 2) meter

jika keliling taman 56 meter

Ditanyakan :

Panjang dan lebar taman ?

Jawab :

Rumus keliling persegi panjang :

K = 2panjang + 2lebar

56 = 2(3x - 6) + 2(x + 2)

56 = 6x - 12 + 2x + 4

56 = 6x + 2x - 12 + 4

56 = 8x - 8

56 + 8 = 8x

64 = 8x

x = 64/8

x = 8

kita subsitusikan x = 8 ke panjang dan lebar taman :

= panjang taman

= (3x - 6)

= (3.(8) - 6)

= (24 - 6)

= 18 meter

= Lebar Taman

= (x + 2)

= (8 + 2)

= 10 meter

jadi, panjang dan lebar taman sebenarnya adalah

panjang 18 meter dan lebar 10 meter

Jawaban Detail ✅

Jawaban (A)


26. Soal Latihan USBN 2017 / 2018 Mapel: Matematika (Peminatan) Satuan Pendidikan: SMA Tahun: 2017 / 2018 Kurikulum: Kurikulum 2013


Kelas : 11
Mapel : Matematika
Kategori : Trigonometri Lanjut, Limit
Kata Kunci : Sudut rangkap, persamaan trigonometri, Limit x mendekati tak terhingga
Kode : 11.2.3 (Kelas 11 Matematika Bab 3 - Trigonometri Lanjut), 11.2.7 (Kelas 11 Matematika Bab 7 - Limit)

Pembahasan :

1) cos 2x - 3 cos x + 2 = 0
=> (2 cos² x - 1) - 3 cos x + 2 = 0
Misal cos x = a
=> (2a² - 1) - 3a + 2 = 0
=> 2a² - 3a + 1 = 0
=> (2a - 1)(a - 1) = 0
=> a = 1/2 atau a = 1

Untuk a = 1/2
cos x = 1/2
Kuadran 1 : x = 60° = π/3
Kuadran 4 : x = 300° = (5/3) π

Untuk a = 1
cos x = 1
x = 0° = 0
x = 360° = 2π

HP = {0, π/3, (5/3) π, 2π}


2) Rumus yang digunakan :
Lim √(ax² + bx + c) - √(px² + qx + r)
x=>~
Jika a > p => ~
Jika a = p => (b - q)/(2√a)
Jika a < p => -~

Lim √(4x² + 6x - 7) - 2x - 3
x=>~

Lim √(4x² + 6x - 7) - (2x + 3)
x=>~

Lim √(4x² + 6x - 7) - √(2x + 3)²
x=>~

Lim √(4x² + 6x - 7) - √(4x² + 12x + 9)
x=>~

a = 4, b = 6, c = -7, p = 4, q = 12, r = 9
Karena a = p maka bisa menggunakan rumus :

(b - q)/(2√a)
= (6 - 12)/(2√4)
= -6/4
= -3/2
= -1 1/2
= -1,5
