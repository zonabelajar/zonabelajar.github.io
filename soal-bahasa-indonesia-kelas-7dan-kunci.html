1. Pecahan yang menyatakan jumlah seluruh (kelas 7dan kelas 8)yang memilih film action adalah....


Kategori Soal : Matematika - Bilangan Pecahan
Kelas : VII (1 SMP)
Pembahasan :
Bentuk umum dari pecahan adalah a\b dengan b ≠ 0. Pada pecahan a\b, a dinamakan pembilang dan b dinamakan penyebut.

Pecahan yang senilai atau sama nilainya dengan a\b dapat diperoleh dengan cara mengali atau membagi pembilang dan penyebut pecahan tersebut dengan bilangan yang sama.
Untuk sebarang pecahan a\b dengan b ≠ 0 berlaku
a\b = (a x m)\(b x m) atau a\b = (a : n)\(b : n) dengan m dan n sebarang bilangan bukan nol.
Contoh : 2\3 = (2 x 5)\(3 x 5) = 10\15 artinya 2\3 dan 10\15 pecahan senilai.
               14\21 = (14 : 7)\(21 : 7) = 2\3 artinya 14\21 dan 2\3 pecahan senilai.

Pecahan a\b dengan b ≠ 0 dapat disederhanakan dengan cara pembilang dan penyebut dibagi dengan FPB dari a dan b.
Contoh : 16\24 = (16 : 8)\(24 : 8) = 2\3 artinya 16\24 disederhanakan menjadi 2\3 dan 8 adalah FPB dari 16 dan 24.

Mari kita lihat soal tersebut. Soal belum lengkap.

40. kelipata 7dan 1000 per 3000​


Jawaban:

a = 1001

b = 7

Un = 2996

a + (n-1)b = 2996

1001 + (n-1)7 = 2996

7n - 7 = 2996 - 1001

7n = 2996 - 1001 + 7

7n = 2002

n = 2002 : 7

n = 286

Sn = 1/2 n ( a+Un )

= 1/2.286 (1001 + 2996)

= 143.3997

= 571571

Jawaban:

571571

Penjelasan dengan langkah-langkah:

Kelipatan 7

Kel.1 = 1001

Kel.n= 2996

b = 7

Un = a + (n - 1)b

2996 = 1001 + 7 × (n - 1)

1995 = 7 × (n - 1)

n - 1 = 1995/7

n - 1 = 285

n = 286

Sn = 1/2 n × (2a + (n - 1)b)

Sn = 1/2 × 286 × (2002 + 285 × 7)

Sn = 143 × (2002 + 1995)

Sn = 143 × 3997

Sn = 571571
