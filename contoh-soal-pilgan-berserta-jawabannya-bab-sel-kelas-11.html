20. tolong kakk besok dikumpull (contoh soal pilgan sma pembahasannya beserta tahun soalnya) ​


Contoh soal SBMPTN Matdas 2013 (kode 326)
Jika diketahui 9ⁿ⁺¹ - 2(9ⁿ) = 14, berapa nilai 27ⁿ

Nilai 27ⁿ = (3³)ⁿ = 3³ⁿ

9ⁿ⁺¹ - 2(9ⁿ) = 14
-->  9ⁿ · 9 - 2(9ⁿ) = 14
-->  (9-2)(9ⁿ) = 14
-->  7(9ⁿ) = 14
-->  9ⁿ = 14÷7
-->  9ⁿ = 2
-->  3²ⁿ = 2

Tentukan nilai 3ⁿ
3²ⁿ = 2
-->  (3ⁿ)² = 2
-->  3ⁿ = √2

Maka nilai dari 27ⁿ, (atau 3³ⁿ) adalah...
3³ⁿ = 3²ⁿ⁺ⁿ = 3²ⁿ × 3ⁿ = 2 × √2
= 2√2

Nilai dari 27ⁿ = 2√2
(xcvi)

35. contoh soal turunan fungsi pilgan beserta caranya


Carilah turunan f'(x) untuk f(x) = (x2 + 2x + 3)(4x + 5)

Pembahasan
Misal :
u = (x2 + 2x + 3)
v = (4x + 5)

Sehingga didapatkan
u' = 2x + 2
v' = 4

Kemudian kita masukkan ke dalam rumus f'(x) = u'v + uv' sehingga turunannya menjadi :
f'(x) = (2x + 2)(4x + 5) + (x2 + 2x + 3)(4) 
f'(x) = 8x2 + 10x + 8x + 10 + 4x2 + 8x + 12 
f'(x) = 8x2 + 4x2 + 10x + 8x + 8x + 10 + 12 
f'(x) = 12x2 + 26x + 22
