Pada tahun 1987, Thomas Knoll, mahasiswa PhD di Universitas Michigan, mulai menulis sebuah program pada Macintosh Plus-nya untuk menampilkan gambar grayscale pada layar monokrom. Program ini, yang disebut Display, menarik perhatian saudaranya John Knoll, seorang karyawan di Industrial Light & Magic, yang merekomendasikan Thomas agar mengubah programnya menjadi program penyunting gambar penuh. Thomas mengambil enam bulan istirahat dari studi pada tahun 1988 untuk berkolaborasi dengan saudaranya pada program itu, yang telah diubah namanya menjadi ImagePro.[1] Setelah tahun itu, Thomas mengubah nama programnya menjadi Photoshop dan bekerja dalam jangka pendek dengan produsen scanner Barneyscan untuk mendistribusikan salinan dari program tersebut dengan slide scanner; "total sekitar 200 salinan Photoshop telah dikirimkan" dengan cara ini.[2]

Selama waktu itu, John bepergian ke Silicon Valley di California dan memberikan demonstrasi program itu kepada insinyur di Apple Computer Inc. dan Russell Brown, direktur seni di Adobe. Kedua demonstrasi itu berhasil, dan Adobe memutuskan untuk membeli lisensi untuk mendistribusikan pada bulan September 1988.[1] Sementara John bekerja pada plug-in di California, Thomas tetap di Ann Arbor untuk menulis kode program. Photoshop 1.0 dirilis pada 1990 khusus untuk Macintosh.[3]

Jawaban:

Cara Menggunakan Selection pada Photoshop CS 2

Selection adalah suatuh teknik dasar yang diperlukan untuk memanipulasi dan pengeditan suatuh image (gambar).

￼

Selection terdiri dari beberapa Tool diantaranya :

Marquee Tool

Lasso Tool

Magic Wand

Color Range

Pen Tool

Type Mask

Alpha Channel

Untuk membuat selection, Kita dapat mengkombinasikan Tool-tool dan perintah seleksi, lalu memodifikasi seleksi yang sudah ada sehingga akan menghasilkan seleksi yang diinginkan.

Berikut adalah perintah-perintah dasar dalam teknik seleksi.

1. Menyeleksi seluruh kertas kerja.

Menyeleksi 1 area penuh pada kertas kerja caranya : Klik Menu Select >> All >> atau tekan Ctrl + A di keyboard.

2. Membatalkan / Menghapus Seleksi.

Klik menu Select >> Deselect atau tekan Ctrl + D di keyboard.

3. Memanggil Seleksi.

Klik menu Select >> Reselect >> atau tekan Ctrl + Shift + D.

4. Membalikkan Seleksi.

Untuk menyeleksi diluar daerah seleksi yang aktif caranya :

Klik Menu Select >> Inverse atau tekan Ctrl + Shift + I.

11. jelaskan langkah-langkah untuk menerapkan anti aliasing pada Photoshop?please jawab besok dikumpul​


Jawaban:

Gunakan salah satu metode berikut. Yang pertama adalah yang paling sulit tetapi memberi Anda lebih banyak kontrol dan dapat menghasilkan hasil yang jauh lebih baik.

Pen Tool - Pilih Pen Tool dan mulailah membuat jalur di sekeliling bentuk aliased. Saya tidak akan menjelaskan cara menggunakan Pen Tool. Periksa opsi kedua jika Anda tidak tahu caranya. Jika tidak, gambar jalur, lalu pergi ke panel Jalur, klik kanan pada jalur yang Anda buat dan pilih 'Buat Pilihan ...' Di jendela sembulan yang muncul, pastikan bahwa bulu diatur ke 0 dan Anti- Kotak centang Aliased dicentang. Kemudian klik OK. Cara lebih cepat untuk membuat pilihan ini jika Anda yakin anti-aliasing dicentang, adalah dengan Ctrl-Kiri-Klik path di panel Paths.

Magnetic Lasso Tool - Beralih ke Magnetic Lasso Tool dengan menahan tombol Lasso Tool sampai semua varian alat muncul, lalu pilih Magnetic Lasso Tool. Di bilah opsi, pastikan Anti-alias dihidupkan dan atur Radius, Kontras, dan Frekuensi masing-masing menjadi 30px, 1%, dan 100 (mainkan dengan nilai yang berbeda jika tidak cocok untuk Anda). Alat ini bekerja dengan cara yang unik, Anda harus mengklik alat di dekat tepi bentuk yang Anda lacak, tetapi jangan tahan tombol mouse. Gerakkan mouse untuk melacak bentuk dengan kursor sebaik mungkin, lalu klik titik jangkar awal untuk menutup loop pilihan.

Perbaiki Tepi - Ini adalah metode yang paling tidak akurat, tetapi masih dapat menghasilkan seleksi anti-alias semi-layak dengan cepat. Pergi ke panel Layers dan Ctrl-Left-Click (Command-Left Click pada Mac OS X) pada ikon layer yang akan dipilih. Ini akan membuat pilihan alias di sekitar bentuk Anda. Kemudian pilih Pilih> Perbaiki Tepi ... melalui menu utama. Di kotak dialog Perbaiki Tepi, atur Kontras ke 40% dan Halus ke 50 dan atur yang lainnya ke 0. Anda juga bisa bermain dengan pengaturan Halus dan Kontras jika itu tidak bekerja untuk Anda.

Bagian 2: Kontrak, Balikkan, dan Potong

Sekarang Anda memiliki pilihan anti-alias dalam bentuk yang sama dengan path, pergi ke Pilih> Ubah> Kontrak ... melalui menu utama dan masukkan 1 px dalam dialog yang muncul. Tekan OK di dialog, dan kemudian klik Ctrl + Shift + I di Windows atau Cmd + Shift + I di Mac untuk membalikkan seleksi. Sekarang tekan tombol Delete di keyboard Anda untuk memotong tepi alias.

Ini harus menghasilkan garis objek Anda yang benar dan anti-alias. Ini akan menjadi satu piksel lebih tipis dari sebelumnya, tetapi dalam banyak kasus itu akan baik-baik saja.
