Penjelasan:

Energi potensial awal 2 elektron

Ep₀ = k.q.q/r₀ = 9.10⁹. (1,6.10⁻¹⁹)²/(2.10⁻¹⁴) = 11,52.10⁻¹⁵ J

Energi potensial akhir 2 elektron

Ep = k.q.q/r= 9.10⁹. (1,6.10⁻¹⁹)²/(5.10⁻¹⁴) = 4,608.10⁻¹⁵ J

Selisih energi potensial

ΔEk = 11,52 - 4,608 = 6,912.10⁻¹⁵ J

Berkurangnya energi potensial ini menjadi energi kinetik untuk 2 buah elektron.

2. Ek = 6,912.10⁻¹⁵ J

2. 1/2.m.v² = 6,912.10⁻¹⁵

m.v² = 6,912.10⁻¹⁵

v² = 6,912.10⁻¹⁵/m

v² = 6,912.10⁻¹⁵/( 9,1.10⁻³¹)

v² = 0.7595604396
