Penjelasan dengan langkah-langkah:

a( 2,-1,3) b(2,2,x)

x= -2

maka b( 2,2,-2)

maka a× b= (2×2+(-1)×2+3×(-2))

= (4-2-6)= (-4),

berarti x= -2 benar

sudut a× (b+c)

b+c= (2-5,-1-2,3+3) = (-3,-3,6)

misal vektor b+c= b

maka a× b / |a| |b| = cos teta

maka a× b= (6×(-3)+(-3)×(-3)+3×6)

= (-18+9+18)

= 9

|a| = √ 6²+(-3)²+3²= √ 36+9+9= √ 54= 3√6

|b| = √ (-3)²+(-3)²+6²= √ 9+9+36= √54= 3√6

maka (3√6)²= 54

maka cos teta = 9/54= 1/6

b + b = 4 - 8
2b = -4
b = -4/2
b = -2

• -1 + c = c - c
-1 + c = 0
c = 0 + 1
c = 1

• d + (-9) = -13 - d
d - 9 = -13 - d
d + d = -13 + 9
2d = -4
d = -4/2
d = -2

a + b + c + d = 1 + (-2) + 1 + (-2)
a + b + c + d = 1 - 2 + 1 - 2
a + b + c + d = -1 - 1 = -2

Penjelasan dengan langkah-langkah:

vektor a = (4, -1)

vektor c = (-2, 1)

vektor d = (-1, -2)

vektor b = (2, 3)

a + c = (4+ (-2), -1+1) = (2, 0)

b + d = (2 + (-1), -2+3) = ( 1, 1)

a + b + c = (4+2+(-2), -1 + 3 + 1) = (4, 3)

a - d = (4-(-1), -1-(-2)) = (5, 1)

b - c = (2 - (-2), 3-1) = (4, 2)

a + 2d = (4 + 2 (-1), -1 +2(-2)) = (2, -5)

b + 2c + 3d = (2+ 2 (-2) + 3 (-1), 3 + 2 (1) + 3 (-2))

= (-5, -1)

a - 2d = (4 - 2(-1), -1-2(-2)) = (6, 3)

3c - 2d = (3(-2) - 2(-1), 3(-2) - 2(-2)) = (-4, -2)

1/2a - d = (1/2(4) - (-1), 1/2(-1) - (-2)) = (3, 3/2)

36. Matematika Peminatan kelas X
