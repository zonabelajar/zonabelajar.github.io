f(x) = (2x² - x - 3)(3x + 2)

f'(x) = 3.6x³⁻¹ + 2.x²⁻¹ - 11x¹⁻¹ - 0
f'(x) = 18² + 2x - 11 (B)

18). f(x) = x⁴ - 4x²
Nilai minimum diperoleh ketika f'(x) = 0
f'(x) = 4x³ - 8x ⇔ 4x(x² - 8) = 0
⇔ 4x(x - √8)(x + √8) = 0
x = 0 atau x = √8 dan x = -√8
f(x) = x⁴ - 4x²
f(0) = 0
f(-√8) = 32
f(√8) = 32
Jadi, nilai minimum = 0 (E)

19). f(x) = x³ - 9x² + 15x, interval -1 < x < 6
1). Titik balik diperoleh ketika f'(x) = 0
f'(x) = 3x² - 18x + 15 = 0
f'(x) = x² - 6x + 5 = 0
⇒ (x - 5)(x - 1) = 0
⇒ x = 5 atau x = 1

Nilai balik:
f(x) = x³ - 9x² + 15x
f(5) = 5³ - 9(5)² + 15(5) = - 25
f(1) = 1³ - 9(1)² + 15(1) = 7

Nilai pada ujung interval:
f(x) = x³ - 9x² + 15x
f(-1) = -1³ - 9(-1)² + 15(-1) = -25
f(6) = 6³ - 9(6)² + 15(6) = -18

Jadi, nilai maksimumnya adalah -25 (B)17) f(x) = (x + 1)(2x - 3)(3x + 2)

18) f(x) = x^4 - 4x^2
f'(x) = 4x^3 - 8x = 0
=> 4x(x^2 - 2) = 0
=> 4x(x + √2)(x - √2) = 0
=> x = 0, x = -√2, x = √2
x = 0 => f(0) = 0^4 - 4(0)^2 = 0
x = √2 => f(√2) = (√2)^4 - 4(√2)^2 = 4 - 4(2) = -4
x = -√2 => f(-√2) = (-√2)^4 - 4(-√2)^2 = 4 - 4(2) = -4
Jadi nilai balik minimumnya = -4 (tak ada di option)

3) f(x) = x^3 - 9x^2 + 15x pd interval -1 < x < 6

15) y = ax^2 + bx + 2
(2, 0) => 0 = a(2)^2 + b(2) + 2
=> 4a + 2b + 2 = 0
=> 2a + b + 1 = 0
=> 2a + b = -1

y = -3x + 6 => m = -3
m = y'
y' = 2ax + b
m = 2a(2) + b = -3
=> 4a + b = -3

Eliminasi
4a + b = -3
2a + b = -1
--------------- --
2a = -2 => a = -1

2a + b = -1
2(-1) + b = -1
b = 1

=> x^2 - 3x + 2 < 0
=> (x - 1)(x - 2) < 0
