Sifat-Sifat Bilangan Berpangkat
a. Sifat Perkalian Bilangan Berpangkat
Jika m, n adalah bilangan bulat positif dan a bilangan bulat maka am x an = am + n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 x 33 = 39!
Jawab:
36 x 33 = ( 3 x 3 x 3 x 3 x 3 x 3) x (3 x 3 x 3)
6 faktor 3 faktor
= (3 x 3 x 3 x 3 x 3 x 3 x 3 x 3 x 3)
(6 + 3) faktor
= 26 + 3
= 29

b. Sifat Pembagian Bilangan Berpangkat
Jika m, n bilangan bulat positif dan m > n dengan a bilangan bulat dan a bukan 0, maka am : an = am - n.
Contoh Soal :
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa 36 : 33 = 33!
Jawab :
36 : 33 = 36
33
= (3 x 3 x 3 x 3 x 3 x 3)
3 x 3 x 3
= (3 x 3 x 3)
(6 - 3) faktor
= 36 - 3
= 33

c. Sifat Perpangkatan Bilangan Berpangkat
Jika a, m, dan n bilangan bulat maka (am)n = am x n.
Contoh Soal:
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa (43)2 = 46.
Jawab:
(43)2 = (43) x (43)
= (4 x 4 x 4) x (4 x 4 x 4)
3 faktor 3 faktor
= 4 x 4 x 4 x 4 x 4 x 4
(3 x 2) faktor
= 43 x 2 = 46

d. Sifat Perpangkatan pada Perkalian
Jika a, b bilangan bulat dan m bilangan bulat positifmaka (a x b)m = am x bm.
Contoh Soal=
Dengan cara menuliskan dalam bentuk perkalian, tunjukkan bahwa (32)3 = 33 x 23.
Jawab:
(32)3 = (3 x 2) x (3 x 2) x (3 x 2)
= (3 x 3 x 3) x (2 x 2 x 2)
3 faktor 3 faktor
= 33 x 23

e. Sifat Perpangkatan pada Pembagian.
Jika a, b bilangan bulat (b bukan 0) dan m bilangan bulat positif, maka (a : b)m = am : bm.
Contoh soal:
Dengan cara menuliskan dalam bentuk pembagian, tunjukkan bahwa (5 : 3)4 = 54 : 34.
Jawab:
(5 : 3)4 = (5/3)4
= (5/3) x (5/3) x (5/3) X (5/3)
= (5 x 5 x 5 x 5) = 54 = 54 : 53
(3 x 3 x 3 x 3) 34

22. bahasa indonesia kelas 7 kurikulum 2013 hal 237


Fabel adalah sebuah karya sastra Indonesia yang menyajikan sifat, sikap, atau watak manusia melalui penggambaran para tokoh hewan atau binatang dalam cerita. Sebagai sebuah karya sastra, teks fabel memiliki beberapa ciri khas. Salah satunya adalah penggunaan bahasa sederhana. Ragam bahasa ini digunakan agar baik isi maupun pesan fabel dapat dimengerti oleh seluruh kalangan terutama anak-anak.

Sebagai sebuah karya sastra, setiap fabel terdiri dari beberapa struktur. Struktur penyusun fabel mencakup orientasi, komplikasi, resolusi, dan koda.

Pembahasan

Pada kesempatan ini, soal menyajikan kita dengan beberapa gambar ilustrasi. Kemudian, kita diminta untuk menyajikan beberapa unsur intrinsik yang sesuai dengan gambar-gambar tersebut. Berikut kakak akan mencoba menjawab pertanyaan tersebut.

A. TOKOH

1. TOKOH -> SINGA

NAMA -> LEO

WATAK -> BAIK HATI

2. TOKOH -> TIKUS

NAMA -> RUBI

WATAK -> BAIK HATI DAN SUKA MENOLONG

3. TOKOH -> PENEBANG POHON

NAMA: LOGO

WATAK -> SOMBONG DAN PEMARAH

B. LATAR TEMPAT -> SEBUAH HUTAN YANG POHONNYA TELAH DITEBANGI

C. LATAR WAKTU -> PAGI HARI

D. LATAR SUASANA -> SEDIH

36. jawaban lks bahasa inggris kls 8 halaman 7 semester 2 kurikulum 2013


Daily Routines

Daily routines atau Daily activities adalah kegiatan yang dilakukan sehari-hari. Kegiatan daily routines dalam Bahasa Indonesia sering juga disebut sebagai kebiasaan/ habit.

Saat kita ingin menyatakan kegiatan sehari-hari/ daily routines, kita perlu menggunakan Simple Present Tense. Simple Present Tense adalah tenses yang digunakan untuk menyatakan kejadian yang berlangsung sehari-hari (Daily routines).

Kata keterangan yang sering digunakan dalam simple present tense adalah:

• Every day

• Every week

• every month

• every year

• once a week

• twice a day

• three times a month

• three times a year

• always

• often

• usually

• seldom

• rarely

• dan lain lain

Pattern yang digunakan dalam simple present tense adalah:

(+) S + V1 atau S + V1s/es

(-) S + do/ does + V1

(?) Do/ DOes + S + V1?

NB:

V1 dan Do untuk subject I, You, They, We

V1s/es dan Does untuk subject He, She, It

Pembahasan

Daily Activities pada teks di atas:

The writer usually wakes up at 5.00 in the morning.He always brushes his teeth, and shaves, and washes his face after alarm wakes up him.He always gets dressed and goes downstair to have breakfast.He always has coffee, cereal and lots of fruits.He always goes work by bus and arrives at his office at 8.00.He finishes his work at 5.00 in the evening and goes home.He and his family always eat dinner in the dining room at around 6.00.They really enjoy watching their favourite TV programs together two or three times a week.He does exercise in the evening.At 9.30, he always goes to bed, reads for a while, and then falls a sleep.
