7. Sebuah warnet kecepatan bandwidth sebesar 0.07 Mbps waktu yang di gunakan untuk kendwonload sebuah file sebesar 2.1 MB adalah 2 menit A waktu yang seharusnya mendwonload file tersebut B besar throughput C selisih bandwidth dan throughput dalam jaringan


A.

waktu download sebenarnya = ukuran file / bandwidth

2,1 Mb / 0,07Mbps = 30 detik

B.

2 menit = 120 detik

waktu  download  typical = ukuran file / throughput

120 detik = 2.1Mb / throughput

throughput = 2.1Mb / 120 = 0.0175Mbps

C.

selisih bandwidth dan throughput

0.07 - 0.0175 = 0.0525Mbps
