Terjemahan Bahasa Jawa:

"Aku seneng tenan ketemu kowe, aku wis gampil kuwi. Aku seneng yen sampeyan iso ngerti apa sing aku isih banget seneng ora entek sampeyan."

Terjemahan Bahasa Indonesia:

"Aku sangat senang bisa bertemu denganmu, aku sudah sangat bahagia ini. Aku senang jika kamu bisa mengerti betapa bahagianya aku tidak bisa menahan kebahagiaanku."

Soal tersebut merupakan soal tentang aksara jawa.

Pembahasan

Soal tersebut merupakan soal Bahasa Jawa yang membahas tentang aksara jawa. Aksara jawa adalah ulisan tradisional yang sering dipakai untuk menuliskan bahasa jawa. Aksara jawa ditulis seperti kita menuls huruf pada umumnya yaitu dari kiri ke kanan. Untuk menulis suatu kalimat, aksara Jawa mempunyai pelengkap berupa pasangan, sandhangan, aksara angka, aksara murda, aksara swara, dan aksara rekan.

32. Kuis ala-ala Jawa Tuliskan aksara-aksara rekan beserta 3 contoh kata​


Nulis aksara jawa

jawaban terlampir

Pembahasan

Aksara rekan yaiku aksara jawa kang digunakake kanggo mbiyantu pemenuhan kebutuhan ing panulisan lan ukara ukara sing asale saka basa liya.

Aksara rekan cacahe ana 5 :

Kha = ꦏ꦳Da = ꦢ꦳Ga = ꦒ꦳Ja = ꦗ꦳Pa = ꦥ꦳

Jlentrehne aksara jawa

1. Khalid = ꦏ꦳ꦭꦶꦢ꧀

Khalid = aksara rekan Kha, aksara La + wulu, aksara Da + pangkon.

2. Dzat murni = ꦢ꦳ꦠ꧀ꦩꦸꦂꦤꦶ

dzat = aksara rekan Dza, aksara Ta (sigeg)murni = pasangan aksara Ma + suku + layar, aksara Na + wulu.

3. Hukum ghibah = ꦲꦸꦏꦸꦩ꧀ꦒ꦳ꦶꦧꦃ

hukum = aksara Ha + suku, aksara Ka + suku, aksara Ma (sigeg)ghibah = pasangan aksara rekan Gha, + wulu, aksara Ba + wignyan.

4. Zakat fitrah = ꦗ꦳ꦏꦠ꧀ꦥ꦳ꦶꦠꦿꦃ

zakat = aksara rekan Za, aksara Ka, aksara Ta (sigeg)fitrah = pasang aksara rekan Fa + wulu, aksara Ta + cakra + wignyan.

5. Fakir miskin = ꦥ꦳ꦏꦶꦂꦩꦶꦱ꧀ꦏꦶꦤ꧀

fakir = aksara rekan Fa, aksara Ka + wulu + layarmiskin = aksara Ma + wulu, aksara Sa (sigeg), aksara Ka + wulu, aksara Na + pangkon.

33. aksara jawa aksaraa jawaa


Jawaban: sedasa

Aksara sa(ꦱ) diberi pepet(ꦼ) menjadi se, Aksara da(ꦢ) legena tetap dibaca da, dan aksara Sa(ꦱ) legena tetap dibaca sa

Ngartekake aksara jawa

ꦱꦼꦢꦱ = Sedasa __________________Pembahasan

Ngartekake aksara jawa iku sing kudu dimangerteni yaiku pasangan lan sandhangane.

ꦱꦼꦢꦱ = sedasasedasa = Aksara Sa + pepet, aksara Da, aksara Sa.

Tembung Sedasa tegese Sepuluh utawa ing basa Indonesia tegese Sepuluh.

Aksara angka :

0 = ꧐ = enol1 = ꧑ = siji = setunggal2 = ꧒ = loro = kaleh3 = ꧓ = tiga = tiga 4 = ꧔ = papat = sekawan5 = ꧕ = lima = gangsal6 = ꧖ = enem = enem7 = ꧗ = Pitu = pitu8 = ꧘ = wolu = wolu 9 = ꧙ = Sanga = sanga10 = ꧑꧐ = sepuluh = sadasa
