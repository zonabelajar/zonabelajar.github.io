14. *Ini soalnya bukan pilihan ganda​


Jawaban:

a. (f o g)(x) = 6x² + 3

b. (g o f)(x) = 12x² + 12x + 4

c. (f o f)(x) = 4x + 3

d. (f o g)(4) = 99

Penjelasan dengan langkah-langkah:

a. (f o g)(x) = f(3x² + 1)

= 2(3x² + 1) + 1

= 6x² + 2 + 1

(f o g)(x) = 6x² + 3

b. (g o f)(x) = g(2x + 1)

= 3(2x + 1)² + 1

= 3(4x² + 4x + 1) + 1

= 12x² + 12x + 3 + 1

(g o f)(x) = 12x² + 12x + 4

c. (f o f)(x) = f(2x + 1)

= 2(2x + 1) + 1

= 4x + 2 + 1

(f o f)(x) = 4x + 3

d. (f o g)(x) = 6x² + 3

(f o g)(4) = 6(4)² + 3

(f o g)(4) = 6(16) + 3

(f o g)(4) = 96 + 3

(f o g)(4) = 99
