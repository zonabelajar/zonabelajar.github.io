3. buatlah soal tik( kelas 7) semester 2 sebanyak 10 berserta jawabannya ?​


MATERI: MICROSOFT OFFICE (WORD)

1. Apa fungsi CTRL + C?

Fungsi CTRL + C (copy) adalah untuk menyalin huruf, file, dsb.

2. Apa fungsi CTRL + V?

Fungsi CTRL + V (paste) adalah untuk menempelkan hal yang sudah dicopy sebelumnya.

3. Apa fungsi CTRL + B?

Fungsi CTRL + B (bold) adalah untuk menebalkan huruf / angka.Contoh: ABC

4. Apa fungsi CTRL + I?

Fungsi CTRL + I (italic) adalah untuk memiringkan huruf / angka.Contoh: ABC

5. Apa fungsi CTRL + U?

Fungsi CTRL + U (underline) adalah untuk memberikan garis bawah dibawah huruf / angka.Contoh: ABC

6. Apa fungsi CTRL + P?

Fungsi CTRL + P menampilkan form untuk print dokumen word. Di CTRL + P juga ada opsi untuk mengatur ukuran kertas yang akan digunakan.

7. Apa fungsi CTRL + N?

Fungsi CTRL + N adalah untuk membuat dokumen baru.

8. Apa fungsi subscript?

Subscript berguna untuk membuat huruf / angka menjadi kecil dan berletak di bawah.Contoh: [tex]X_{2}[/tex]  (angka "2" berada di bawah)

9. Apa fungsi superscript?

Superscript berguna untuk membuat huruf / angka menjadi kecil dan berletak di atas.Contoh: [tex]X^{2}[/tex]  (angka "2" berada di atas)

10. Sebutkan guna Microsoft Word!

Membuat, mengedit dokumen.

33. Buat lah 15 soal tentang (TIK) perangkat lunak aplikasi, fitur dan bagian² tentang keyboard beserta jawaban nya ..​


Soal 1

Apa yang dimaksud dengan perangkat lunak aplikasi?

Jawaban:

Perangkat lunak aplikasi adalah perangkat lunak yang dirancang untuk menyelesaikan tugas tertentu, seperti mengetik, membuat presentasi, atau bermain game. Perangkat lunak aplikasi biasanya dibeli atau diunduh dari internet.

Soal 2

Apa saja jenis-jenis perangkat lunak aplikasi?

Jawaban:

Jenis-jenis perangkat lunak aplikasi dapat dikelompokkan berdasarkan fungsinya, seperti:

Perangkat lunak produktivitas, seperti Microsoft Office, Adobe Photoshop, dan Google Chrome.

Perangkat lunak hiburan, seperti game, pemutar musik, dan pemutar video.

Perangkat lunak pendidikan, seperti software pembelajaran bahasa, software pembelajaran matematika, dan software pembelajaran komputer.

Perangkat lunak bisnis, seperti software akuntansi, software manajemen proyek, dan software manajemen database.


Soal 3

Apa saja fitur-fitur yang umum terdapat pada perangkat lunak aplikasi?

Jawaban:

Fitur-fitur yang umum terdapat pada perangkat lunak aplikasi antara lain:

Fitur antarmuka pengguna, seperti menu, tombol, dan ikon.

Fitur pengeditan, seperti cut, copy, paste, dan undo.

Fitur format, seperti font, ukuran, dan warna.

Fitur kolaborasi, seperti berbagi file dan bekerja bersama.

Fitur keamanan, seperti proteksi kata sandi dan enkripsi data.


Soal 4

Apa saja bagian-bagian yang terdapat pada keyboard?

Jawaban:

Keyboard terdiri dari beberapa bagian, yaitu:

Bagian tombol, yaitu bagian yang digunakan untuk mengetik huruf, angka, dan simbol.

Bagian tombol fungsi, yaitu bagian yang digunakan untuk menjalankan perintah khusus.

Bagian tombol navigasi, yaitu bagian yang digunakan untuk menggerakkan kursor.

Bagian tombol kontrol, yaitu bagian yang digunakan untuk menjalankan perintah umum, seperti enter, delete, dan backspace.


Soal 5

Apa fungsi dari tombol enter?

Jawaban:

Tombol enter berfungsi untuk mengakhiri baris teks atau menjalankan perintah.


Soal 6

Apa fungsi dari tombol delete?

Jawaban:

Tombol delete berfungsi untuk menghapus karakter di depan kursor.


Soal 7

Apa fungsi dari tombol backspace?

Jawaban:

Tombol backspace berfungsi untuk menghapus karakter di belakang kursor.


Soal 8

Apa fungsi dari tombol panah?

Jawaban:

Tombol panah berfungsi untuk menggerakkan kursor ke arah yang diinginkan.

Soal 9

Apa fungsi dari tombol home?

Jawaban:

Tombol home berfungsi untuk menggerakkan kursor ke awal baris.

Soal 10

Apa fungsi dari tombol end?

Jawaban:

Tombol end berfungsi untuk menggerakkan kursor ke akhir baris.


Soal 11

Apa fungsi dari tombol page up?

Jawaban:

Tombol page up berfungsi untuk menggerakkan kursor ke atas satu halaman.

Soal 12

Apa fungsi dari tombol page down?

Jawaban:

Tombol page down berfungsi untuk menggerakkan kursor ke bawah satu halaman.

Soal 13

Apa fungsi dari tombol tab?

Jawaban:

Tombol tab berfungsi untuk indentasi teks.


Soal 14

Apa fungsi dari tombol caps lock?

Jawaban:

Tombol caps lock berfungsi untuk mengaktifkan mode huruf kapital.


Soal 15

Apa fungsi dari tombol shift?

Jawaban:

Tombol shift berfungsi untuk mengetik huruf kapital dan simbol.
