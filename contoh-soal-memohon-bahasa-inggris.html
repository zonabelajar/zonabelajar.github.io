9. ungkapan mengusulkan dan memohon dalam bahasa inggris


kelas : VII
pelajaran : Bahasa Inggris
kategori : expression
kata kunci : giving for help, asking for help

pembahasan:
ungkapan mengusulkan di sini terlalu general, karena ada bermacam-macam ungkapan. Seperti ungkapan mengusulkan bantuan dan mengusulkan opini.

contoh ungkapan mengusulkan bantuan:
1) May I help you?= bolehkah saya membantu kamu?
2) Can I help you? =dapatkah saya membantu kamu?
3) Would you mind if I help you? = apakah kamu keberatan jika saya membantu kamu?
4) Shall I help you? = haruskah aku membantu kamu?

contoh asking for help
1) Can you help me? = dapatkah kamu membantu saya?
2) Would you mind if you help me? = akankah kamu keberatan jika membantu saya?
3) Could you help me, please? = dapatkah kamu membantu saya?

17. mohon bantuannya untuk menjawab soal nomor 27 saya memohon dengan sangat​


Perhatikan Segitiga SQR.

Sudut QRS = 30°, Sudut SQR = 60°,

maka Sudut QSR = 180° - 30° - 60° = 90°

Sudut PSQ = 180° - 90° = 90°

Sudut PQS = 180° - 90° - 45° = 45°

Karena Sudut SPQ dan Sudut PQS sama, maka panjang SP = panjang QS = 4 cm

Perhatikan ΔQRS, siku-siku di S

sin ∠R = SQ / QR

sin 30° = 4 / QR

1/2 = 4/QR

QR = 4 × 2 = 8 cm (C)
