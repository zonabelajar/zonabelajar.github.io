32. The correct arrangement is...


Jawaban:

B. 4256371

Penjelasan:

Please correct me if I make mistake.

Jawaban :

Please correct me if I make mistake.(B) 4-2-5-6-3-7-1

PEMBAHASAN

Kalimat di atas merupakan salah satu ungkapan ask something (meminta sesuatu). Dimana pada kalimat di atas, yang diminta adalah "please correct me" hal ini dilakukan saat "if I make mistake".

Adapun arti dari ungkapan di atas yaitu :

Tolong koreksi saya jika saya melakukan kesalahan.

Ask something adalah ungkapan yang digunakan ketika ingin meminta sesuatu, entah itu berupa bantuan ataupun sesuatu hal. Berikut ini beberapa contoh ungkapan nya :

Please help me ....Can I have...?Please put these books on the table.May I have..?Dan sebagainya.

Grammar dari pilihan ganda :

(A) Please me correct I if make mistake ❌

Kata me merupakan pronoun object. Dimana pronoun yang berfungsi sebagai object letaknya harus di belakang verb. Sedangkan disini letaknya ada di depan verb "correct". So, ini kurang tepat.

(C) If me make mistake please correct I ❌

Di pilihan (C) ini juga letak dari me ada di depan kata kerja "make". Seharusnya terletak di belakang verb.

(D) If I correct me please make mistake ❌

Ketika di artikan "Jika saya mengoreksi saya, tolong buat kesalahan".Dari segi arti sudah sangat tidak masuk akal.

(E) If make mistake I please correct me ❌

Dari segi arti, kalimat ini juga tidak masuk akal."Jika membuat kesalahan saya tolong koreksi saya".

So, jawaban yang tepat adalah (B)

Please correct me If I make mistake.Tolong koreksi saya jika saya membuat kesalahan.
